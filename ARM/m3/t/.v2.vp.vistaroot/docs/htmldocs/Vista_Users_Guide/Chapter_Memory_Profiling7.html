Memory Statistics Plot Tab

To plot the memory statistics data:

  1. Select a process (or processes).
  1. Do one of the following:
  • In the right-mouse-button context, select Add to Plot.
  • Click the Plot icon:

The selected process is displayed in the graph.

Figure 13‑2. Memory Statistics Plot Tab

  1. To view the selected process in the context of the desired graph type, (Total Potential Memory Leak, Number of Potential Memory Leaks, or Maximum Allocation) select the graph from the drop-down list.

Figure 13‑3. Graph Type Drop-down List


  Note

It is sometimes necessary to click the Refresh icon to obtain the updated results.

Clearing the Graph

To clear the graph in the Memory Statistics Plot tab, do the following:

  • Select Plot > Clear All From Plot
