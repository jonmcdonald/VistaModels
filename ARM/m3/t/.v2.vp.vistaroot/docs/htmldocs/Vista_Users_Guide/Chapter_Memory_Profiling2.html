Total Potential Memory Leak

At any time during simulation, you can view the total of the potential memory leaks for a specific process. A potential leak is the amount of allocation that up to the current simulation time was not freed. Please note the following:

  • Vista will not display ALL memory allocations made until the current time; only those that were not freed will be shown.
  • Past memory leaks that are displayed at a given time (in the plot histogram), may not display at a later simulation time, as they may have been freed by that later time.
