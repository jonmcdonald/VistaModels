Creating Virtual Prototype for Windows

Creating VP-for Windows using an external Makefile

To create a Virtual Prototype for Windows using an external makefile, do the following:

  • Instead of using vista_vp_c++, use: vista_vp_cl. This command accepts the same arguments as vista_cc; it is a gcc style driver for MS compiler.
  • To create static libraries (archives) use: vista_create_archive -cl mylib.lib object1.obj object2.obj…

  Note

Make sure the objects' extension is: .obj, and the static library's extension is: .lib.
