Navigating Within the Simulation Time

You can change the displayed “window” (over time) of data, in several ways.

To change the scope displayed in the graph pane:

  • Customize the zoom level by using the zoom icons, the View menu, or the left mouse button (press-drag-release), or by adjusting the Start and End endpoints in the toolbar.
    The summary data is updated to reflect the new scope.
  • Move the mouse over the graph to view plotted values at a particular point in time.
  • Use the navigation buttons to display the desired portions of the run: Scroll to 0, Scroll left, Scroll right, and Scroll to End.
  • You can mark two points in the graph pane and see the Delta values (x and y them) by using the right-mouse-button: press/drag/release.
  • Changing the “Sampling Interval” - Click the “up” or “down” arrows to increase or decrease the sampling interval by 10 or explicitly specify the interval's size in the “Sampling Interval” field.
