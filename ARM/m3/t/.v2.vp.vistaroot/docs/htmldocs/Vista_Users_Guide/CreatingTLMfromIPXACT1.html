IP-XACT Configuration
of ARM Component

Vista enables configuration of a predefined generic component with information extracted from the IP-XACT description generated by ARM ADR tool. Currently only the NIC301 MB generic model is supported. The following sections explain how to create a TLM 2.0 model from the IP-XACT RTL description and describe the types of model parameters supported, other models that are generated, any other operations that can be performed on the model and the directory structure created by the ARM ADR tool as a result of generating IP-XACT description.
