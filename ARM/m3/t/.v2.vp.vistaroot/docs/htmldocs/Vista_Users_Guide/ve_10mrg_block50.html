Edit

Table 9‑5. Block Diagram Edit Menu Items 

Item

Description

Undo

Ctrl+Z

Undoes the last editing operation or operations in the current window buffer, depending on the number of times you invoke the command.

Redo

Ctrl+Y

Redoes the last undone editing operation or operations in the current window buffer, depending on the number of times you invoke the command.

Cut

Ctrl+X

Cuts the selected text/items from the editing window and places it onto the clipboard.

Copy

Ctrl+C

Copies the text/items selected in the editing window and places it onto the clipboard.

Paste

Ctrl+V

Pastes the contents of the clipboard into the textual window, starting from the position of the cursor (textual insertion marker).

Delete Del

Deletes the selected items from the editing window.

Find

Ctrl+F

Searches for the entered textual string.

Find Next

Ctrl+G

Searches for the next occurrence of the same string.

Replace

Ctrl+R

Searches for the entered textual string and replaces it with the one you specify.

Select All

Selects all the text/items currently displayed in the window.

Array

Enables the creation of an array from the selected graphical objects (horizontal, vertical or diagonal list, or matrix of the horizontal and vertical)

Rotate

Opens a dialog for specifying direction and angles of rotation

Align

Aligns selected graphic elements and text boxes in the graphic editors.


Array

(Edit > Array or right-click on selection)

Used to create an array of the selected graphical object/s (horizontal, vertical or diagonal list, or matrix of the horizontal and vertical).

Align

(Edit > Align, or right-click in the graphic editors)

Used to align selected graphic elements and text boxes in the graphic editors.

The following are the Align options:

  • Texts/Objects Checkboxes

Specify which elements you want to align: graphic elements (Objects), text boxes (Texts), or both.

  • Snap to Grid Checkbox and Resolution Definition

If you are only aligning text boxes, click Snap to Grid to align them with the window grid. Specify the level of grid point resolution to employ using the Number of Grids to Snap field. For example, if you specify 7 in the Number of Grids to Snap field, all text boxes are aligned with the closest grid line that is a multiple of 7.

  • Align to

Specify where in the window the selected text and/or graphic elements should be aligned: Top, Bottom, Left, or Right.

  • Align to Object Edges

Specify which edge of a the selected graphical element(s) should be used for alignment: Top or Bottom.

  • On Grid Horizontal

If only text boxes are selected for alignment, you can click On Grid to align them to grid lines in a direction perpendicular to the direction of alignment.
