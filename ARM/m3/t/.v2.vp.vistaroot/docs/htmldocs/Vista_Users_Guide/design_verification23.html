Displaying the Function Call Stack

You can view the contents of the call stack from any of the following locations:

  • In the Simulation tool bar; when you click the P icon to the right of this list (or, alternatively, choose the Simulation > Print Stack command), the entire contents of the stack is displayed in the Message pane.
  • In the Simulation Control window (see "Using the Simulation Control Window").

As indicated, choosing the Simulation > Print Stack command (or clicking the P icon adjacent to the stack list) displays in the Message pane the entire contents of the current function call stack. Each line specifies (in gdb format) the information for a frame, including:

  • frame number
  • an address in memory
  • the name of the function
  • the name of the file
  • the line number

The current frame is marked by an arrow, =>.
