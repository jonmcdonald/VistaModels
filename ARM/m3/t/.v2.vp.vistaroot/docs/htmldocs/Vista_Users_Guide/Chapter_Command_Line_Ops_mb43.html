save_tlm_library

Saves all models in a library to either the library physical path or to the path specified in the
-directory option.

Syntax

save_tlm_library [-h | -help] [-usage] [{-d | -directory} directory] library_name

Arguments
Common (Optional)
  • [-h | -help]
Provides help.
  • [-usage]
Displays the syntax for this command.
Optional
  • [{-d | -directory} directory]
Target directory of the library.
Required
  • library_name
Name of the library to be saved.
