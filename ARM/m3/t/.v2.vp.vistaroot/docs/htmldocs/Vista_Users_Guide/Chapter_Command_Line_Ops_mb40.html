report_instance_activity

Reports transaction activity statistics for a specified instance in a specified database.

Syntax

report_instance_activity [-help] [-usage] simulation_dir instance_path output_file

Arguments
Common (Optional)
  • -help
Provides help.
  • -usage
Displays the syntax for this command.
Required
  • simulation_dir
Simulation database object
  • instance_path
Hierarchical instance path, for example, Top.dma_inst
  • output_file
Target output file path
