vista_dump_events

Creates a report of the events that occurred during analysis. (See "Viewing Analysis Reports")

Syntax

vista_dump_events [-from from_time] [-kind kinds] [-name name-pattern] [-to to_time]
[-units time_units] simulation_directory

Arguments
Optional
  • -from from_time
Limits the start time of the report from the specified time. If this option is not specified, the report starts from 0.
  • -kind kinds
Limits the reported object kind/s only to those specified.
Kind options include:
initiator(_socket)
target(_socket)
attribute, method)
To concatenate multiple kinds use the following format:
initiator_socket | method
  • -name name-pattern
Limits reported names to objects that fit the specified pattern.
  • -to to_time
Limits the end time of the report to the specified time. If this option is not specified, the report ends at the end of the simulation.
  • -units time_units
Specifies the time units. Options are fs, ps, ns, us, ms, s.
Required
  • simulation_directory
The simulation database path
