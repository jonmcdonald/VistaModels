Virtual Prototyping

This version of Vista contains functionality for creating Virtual Prototypes.

A Virtual Prototype File is a stand-alone executable, which can be used to invoke simulation of a design created with Vista. Virtual Prototypes contain the simulation executable, any shared libraries it depends upon, Vista runtime files needed to run the simulation, and the following optional items specified by the Virtual Prototype creator:

  • a Vista parameters file for the created design
  • additional files and directories needed to run the simulation
  • environment variable settings needed to run the simulation
  • command-line arguments to be passed to the simulation executable
  • working directory for running the simulation executable

Virtual Prototypes are created using either the Virtual Prototype Dialog in the Vista GUI (Creating a Virtual Prototype Using Vista GUI) or the vista_create_vp command (Creating a Virtual Prototype Using vista_create_vp). “Virtual Prototype Runtime Environment” describes what happens when a Virtual Prototype is invoked.
