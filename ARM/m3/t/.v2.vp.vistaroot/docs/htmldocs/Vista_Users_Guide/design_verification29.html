Enabling/Disabling Timing in Simulation

You can statically or dynamically disable the consumption of time and power.

In order to run only the PV part of the design (LT mode), include the command disable_T(); in your code, or specify the command at the Vista prompt at any time: p disable_T().

The enable_T() command reverts to normal, where the T component of the design is included (AT mode). The is_T_enabled() function returns true if the simulation runs in AT mode and false in LT mode


  Note

These functions are also very useful in debugging the design.
