Compiling a Project

Compiling a project involves creating objects from each of the source files that are referenced by this project.

Vista Compiler

By default, on Linux/Unix, Vista compiles source files using version 4.3.4 of the gcc compiler. Mentor Graphics supplies the gcc compiler for use with Vista under public license. Modifications to the compiler introduced by Mentor Graphics for use with Vista are provided, together with the source files of used packages.

For details about the other compiler options, see "Environment Variables for Controlling Behavior".

Preparing a Makefile

Vista supports the following commands in a makefile:

  • prepare
  • clean
  • all
  • full
  • full-rebuild

Running Compilation

Compilation analyzes a .c or .cpp source file and generates an object (.o file) for it.

When you compile source files using Vista's interface, you essentially run the following g++ command: 

     g++ -gstabst -DVersion=1 -DUNIX -I<include_path> -o<output_directory> 
            -c <source>.cpp

Note that Vista currently uses only the following g++ options.

Table 10‑1. g++ Options 

Option

Definition

-g

Specify delay-related information in a parameterized format. (In the context of Vista, this argument is unnecessary since the application knows what is relevant.)

-D

Specify variables for preprocessor definitions to enable conditional compilation. Making this setting changes the behavior of execution for different platforms. Within Vista, this setting is made using the Project Settings > Compilation tab.

-I

Specifies one or more “include paths” to directories containing external header files used by the current project. Within Vista, this setting is made using the Project Settings > Include Paths tab.

-o

Specifies the output directory. Vista determines this by itself.

-c

Specifies the source file to be compiled.


Note that, by default, source files suffixed “.c” are compiled as C files instead of as C++ files. However, you can configure your project settings so that files having the suffix “.c” will be compiled as C++ files. To do so, in the Project > Settings > General tab, select the “Always compile in C++ mode” checkbox.

To run compilation of one or more source files:

  1. Select the source file(s) in the browser.
  1. Choose the Build > Compile command or click the Compile icon —
