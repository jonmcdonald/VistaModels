About QEMU

QEMU is a generic and open source fast machine emulator which can run the targeted operating system and its applications on a virtual machine in which the host and target CPU's may be different, i.e QEMU can run Linux OS and programs made for one machine (e.g. an ARM board) on a standard PC architecture.

QEMU uses dynamic translation (Just in Time compilation) which achieves very good emulation performance. It is commonly used for the following purposes:

  • Run one operating system on another, such as Windows on Linux or Linux on Windows.
  • Debugging - QEMU is a very useful debug platform; since a virtual machine can be easily stopped, and its state can be inspected, saved and restored.
