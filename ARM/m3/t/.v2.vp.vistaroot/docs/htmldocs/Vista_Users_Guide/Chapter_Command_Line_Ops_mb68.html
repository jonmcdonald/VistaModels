Vista Debugging Commands

Name

Vista debugging commands:

abreak, bbreak, tibreak, pbreak, next_cycle, next_delta, next_process, switch_to_process, add_watch, save_breakpoints, load_breakpoints.

Synopsis

abreak

bbreak

tibreak time-specification

pbreak process-path

next delta

next time

next process

switch_to_process process-path

add_watch expression-or-path

save_breakpoints filename

load_breakpoints filename

Description

The following Vista-specific debugging commands are supported by the Vista simulation prompt:

-- abreak

Break after elaboration.

-- bbreak

Break before elaboration.

-- tibreak time-specification

Break at the specified time.

-- pbreak process-path

Break at the specified process.

-- next delta

Continue till the end of the delta cycle.

-- next time

Continue till the end of the time cycle.

-- next process

Go to the next process.

-- switch_to_process process-path

Switch to the specified process.

-- add_watch expression-or-path

Adds the specified expression or design path to the Watch folder.

-- save_breakpoints filename

Saves breakpoints in the specified file.

-- load_breakpoints filename

Loads breakpoints saved in the specified file.
