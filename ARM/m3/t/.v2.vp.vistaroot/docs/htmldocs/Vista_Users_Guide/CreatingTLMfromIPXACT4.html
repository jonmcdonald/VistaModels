NIC301 Model Parameters

The following NIC301 parameters are supported (either totally or in-part). These parameters are initialized with default values that can be changed after the TLM model has been created to further configure its functionality.

  • AXI slave port:
  • buffering (outstanding in Vista)
  • AXI master interface:
  • buffering (outstanding in Vista)
  • APB master port:
  • buffering (outstanding in Vista)
