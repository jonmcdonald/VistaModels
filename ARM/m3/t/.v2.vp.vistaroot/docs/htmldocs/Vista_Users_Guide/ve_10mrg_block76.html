Colors

Figure 9‑23. Block Diagram Options - Colors Pane

The Colors pane enables you to specifies the default fill and bordering line colors displayed for the Component, Hierarchical Channel, and Signals graphic elements in a block diagram.

Next to each of these items, clicking on the ellipses (...) opens a dialog box list in which the color is defined for that specific item. After you OK the displayed list, a box next to the item is displayed in the selected color.

When you make these settings with this Options panel, you define the default settings for each subsequent block diagram that will be created. Note, however, that you can override these default settings locally by right-clicking on the element and selecting the Colors command from the popup menu.
