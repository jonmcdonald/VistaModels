View Power Distribution Results

Upon selecting the top of the design (or sub-tree unit) and adding it to the graph (double clicking it or selecting the Add to Graph option from the RMB popup menu), the Power Distribution analysis view opens. Specific colors for all module instances that reside under the top are automatically be set and colored areas (one on top of the other) are shown. In addition, the regular power view of the selected top (or sub-tree module) opens above its power distribution view.

  • Power Distribution is selected only if the selected module is hierarchical and instantiates other modules.
  • A Color fills the specific area of the contributing module instance (unlike current line specification).
