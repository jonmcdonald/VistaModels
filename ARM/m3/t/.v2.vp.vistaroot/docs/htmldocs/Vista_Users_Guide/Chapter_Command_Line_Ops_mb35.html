parse_vhdl_source

Parses the VHDL source files into the Vista model builder.

Syntax

parse_vhdl_source [-help] [-usage] source_files library_name

Arguments
Common (Required)
  • -help
Provides help.
  • -usage
Displays the syntax for this command.
Required
  • source_files
VHDL source file path
  • library_name
Target HDL library name
