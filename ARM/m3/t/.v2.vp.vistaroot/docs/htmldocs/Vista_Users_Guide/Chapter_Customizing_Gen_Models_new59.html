Modifying Generic Configuration and Updating Tables

If a model port is added or renamed, you might wish to automatically update the various parts of the specification (for example, registers or policies). You also might want to use the original generic configuration settings that came with this model-type definition. To do one or both of these tasks:

  1. Click the Update Tables from Template toolbar icon or select the menu item in the Tools menu.

The Update from Template dialog appears.

Figure 8‑17. Restoring Generic Configurations from the Template

  1. (Optional) Modify the model's generic configuration, as needed.
  1. Click OK.

The modifications are saved, and any necessary updates are made to the model-specification tables.


  Note

Not all generic models have generic configurations.
