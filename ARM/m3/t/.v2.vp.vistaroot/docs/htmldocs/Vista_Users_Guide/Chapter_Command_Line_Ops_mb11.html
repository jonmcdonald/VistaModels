configure_model_from_ipxact

Create a TLM2.0 component based on an IP-XACT description and saves the created component.

Syntax

configure_model_from_ipxact template_path model_name ipxact_root_path output_library [options]

Arguments
  • template_path
Path to the generic model.
Currently only Nic301 is available: /TLM/Generic_Library/Nic301
  • model_name
Name of the ARM top component created by the ARM ADR tool
  • ipxact_root_path
Full path to the root of the directory tree created by the ARM ADR tool that contains the IP-XACT description for the created component:
…/<project_name>_RTL/logical/component_name
or
…/project_name_RTL/logical
  • output_library
Name of Vista library in which to create the TLM model
  • options
All optional arguments accepted by the create_tlm_model command, in particular:

-generate_kind pvt | pv

-overwrite 1 | 0

-vista_project — name of Vista project in which to add the model

-generate_ipxact 1 | 0

-ipxact_vendor — vendor name to use in generating the IP-XACT description

-ipxact_version — version string to be use in generated IP-XACT description
