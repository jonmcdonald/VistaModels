Printing

As you work in Vista, you can print all or part of your graphical and textual units, and the results of simulation and code generation. This section describes the printing of graphical units, while instructions related to printing of results is covered in the appropriate sections of this User's Guide.

To print a schematic —

  1. If you need to adjust your printer setup, click File > Printer Setup, and use the dialog box that appears.
  1. Make the diagram's window active.
  1. Click File > Print or click the Print command icon and use the dialog box now displayed.
