Bus Definition

The Bus Definition created for a MB protocol contains the following elements:

  • VLNV - name: the name of the protocol (ex. AHB, APB).
  • directConnection = true | false

If the bus is addressable, then only non-mirrored to mirrored connections are allowed and directConnection=false, otherwise directConnection=true.

  • isAddressable = true if the protocol contains “bus_master” or “bus_slave” attribute, else false.
