reload_tlm_library

Loads all models, by .mb file, from the library physical path or path specified in the -directory option to this library.

Syntax

reload_tlm_library [-h | -help] [-usage] [{-d | -directory} directory] library_name

Arguments
Common (Optional)
  • [-h | -help]
Provides help.
  • [-usage]
Displays the syntax for this command.
Optional
  • {-d | -directory} directory
Path to model directory.
Required
  • library_name
Name of the library to be reloaded.
