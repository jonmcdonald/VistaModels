Creating a New sc_main

Vista allows you to automatically create an sc_main method template that can execute your design promptly.

To create a new sc_main:

  1. Expand the header file (.h) of the top sc_module of the design.
  1. Select the icon 3-colored triangular icon for the class

In the tool bar the sc_main command icon is now selectable

  1. Click this icon, or choose the New sc_main command from the combo-box list next to this icon or from the Projects.
  1. In the New File dialog box now displayed, specify the name, directory, and project name for the new sc_main source file.

When you perform these operations, the resulting sc_main instantiates the top-of-design module, from which you can then build your entire design tree.
