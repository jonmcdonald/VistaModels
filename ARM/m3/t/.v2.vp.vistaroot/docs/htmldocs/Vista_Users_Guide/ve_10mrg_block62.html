Filling and Unfilling Elements

You can fill any closed two-dimensional graphic element so that its color is cyan instead of white.

If you drew a single, closed geometric figure using two or more graphic elements (lines or arcs), you can fill it like a single graphic element. As long as it remains filled, it is treated as a single graphic element.

To fill:

  1. Select the element or elements to be filled.
  1. Click the Fill icon: .

To unfill:

  1. Select the element or elements to be unfilled.
  1. Click the Unfill icon: .

The cyan disappears. If the filling grouped together a number of graphic elements, they become individual graphic elements again.
