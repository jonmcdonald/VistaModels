QEMU in Vista

Vista integrates generic fast ISS (Instruction Set Simulator) processors models which are based on QEMU sources - public domain sources (GNU LGPL).

Vista introduces generic ISS models for a number of processors architectures:

  • ARM
  • PPC\MPC
  • MIPS

More architectures are added to the QEMU generic library continuously.

The fast ISS QEMU based models are fully SystemC TLM2.0 compatible and are suitable for:

  • Initial Platform bring up (Architectural Exploration)
  • Platform Performance Analysis
  • Software debugging and analysis (Virtual Prototype)
  • OS bring up

  Note

Any platform may instantiate an arbitrary number of processors making the QEMU solution ideal for multi core platforms, a user may either instantiate a multi core model (e.g. ARM11 MP Core), or create a custom multi core\processor environment according to his requirements, Vista QEMU based modeling scheme is flexible and may support either.
