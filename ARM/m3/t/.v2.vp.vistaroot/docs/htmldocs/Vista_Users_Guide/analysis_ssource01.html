Analysis Graphs and Reports

Vista simulation data can be analyzed for timing and power. During TLM simulation, a transaction-level database is automatically generated. The generated database contains all the transactions and power activity from the simulation, and it can be stored for review in a “dump” file.

The analysis tool set allows you to:

  • visualize transactions and variables in various channels and instances
  • analyze throughput and latency metrics of various resources in the system
  • analyze power distribution
  • compare side-by-side results of two or more simulations
  • begin analyzing partial data during the course of a simulation session

Vista Analysis can be run via the user interface (Tools > Open Analysis), or via the command vista_analysis.

The results generated by the analysis can be displayed via powerful graphing functions in Vista, and/or output in detailed and comprehensive ASCII text reports via batch commands.
