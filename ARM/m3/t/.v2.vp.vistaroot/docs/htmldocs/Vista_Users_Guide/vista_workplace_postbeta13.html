Vista Console Tab

The Console tab displays:

  • Modeling-related messages (after Save, Save & Generate or Add to Project commands)
    Echoes in equivalent command-line form all actions done via the modeling tabs.
  • Input area for the Vista command-line interface (see "Batch and Command-Line Options") - complete scripts can be run, or specific commands can be issued: for example, to add a port to a model.
  • Log of all Standard Output text

One or both of the following text logs can be generated:

  • Commands only
  • Commands and standard-output text
