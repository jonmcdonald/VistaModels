TLM Control Commands

Set Timing Mode

mon tlm set-timing-model {AT | LT}

Sets current TLM timing simulation model to AT or LT mode.

Show Timing Mode

mon tlm show-timing-model

Displays current TLM timing simulation model mode.

Set DMI Status

mon core set-dmi {ON | OFF}

Sets DMI status of core.

If enabled, the simulation core uses TLM DMI protocol.

This command can be used in order to trace transactions in a wave without fully switching to AT simulation mode.


  Note

When DMI is set to off (transactions access peripherals through the system bus), it is possible to view transaction data and starting times in the Vista Waveform while simulating in LT (fast) mode.

Show DMI Status

mon core show-dmi

Displays current DMI status of core. If ON, the simulation core uses the TLM DMI protocol.
