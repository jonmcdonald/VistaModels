Restarting Simulation from the Beginning

The Simulate > Resimulate command lets you perform simulation on a design from the start (time 0), while retaining all breakpoints and signals in the waveform window. When you invoke this command, the Simulation dialog box is displayed, so that you can enter different values for any of the fields there (see option in the Trace Script tab).

Besides using the menu command, you can also invoke this command using the Resimulate icon displayed in the Simulate tool bar:


  Note

Before invoking the Resimulate command, you should save any current simulation data that you may want later - waveform windows for example - because the Resimulate command will make this data unavailable unless it was previously saved.
