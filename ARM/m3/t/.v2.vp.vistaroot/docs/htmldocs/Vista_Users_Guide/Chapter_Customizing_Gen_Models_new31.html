ARM 11 MP Core

Feature List
  • AXI Interface
  • ARM v6 architecture Instruction Set
  • Thumb Instruction Set
  • Floating Point
  • MMU
  • L1 Cache
  • Interrupts
  • Snoop Controller (multi core designs)
Interface Ports
  • Initiator_m0 - AXI bus master port
  • Initiator_m1 - AXI bus master port
  • n_irq - Active low interrupt signal, one signal for each core
  • n_fiq - Active low fast interrupt signal, one signal for each core
  • n_reset - Active low reset signal, one signal for each core
Generation Parameters (Generics)
  • NUM_CPU - Number of ARM11 CPU involved
CPU Specific Parameters
  • dcache_size — L1 data cache size. Default is 32K. The parameter specifies the data cache size in bytes. The size must be an integer which is a power of two. You may configure the dcache_size to be of zero size in order to disable the data cache functionality.
  • icache_size — L1 instruction cache size. Default is 32K. The parameter specifies the instruction cache in bytes. The size must be an integer which is a power of two. You may configure the icache_size to be of zero size in order to disable the instruction cache functionality.
