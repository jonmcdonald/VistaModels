TLM Abstraction Definition

The TLM Abstraction Definition created for a MB protocol contains the following elements:

  • VLNV - name: <bus-definition_name>_tlm20 (ex. AHB_tlm20).
  • busType - reference to the bus definition (VLNV).
  • ports - one single port element that defines the protocol name as logicalName and a transactional element. The transactional element specifies if the port is an address (if the protocol contains address information), a data (if the protocol contains data information), as well as its presence, service initiative and service typeName, when referenced from a master, slave or system type bus interface.
