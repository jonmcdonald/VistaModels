Using Vista TSV with NC

If you intend to utilize Vista's Transaction Sequence Viewer in conjunction with NC simulation, you must specify Vista's -tsv option when compiling. How to do this varies depending on the wrapper script that you will run.

vista_ncsc_run

If you will be running this wrapper script to perform unified compilation, elaboration, and simulation, in the command line with which you invoke the script you must include NC's -Wcxx option to indicate that Vista's -tsv option is to be used, as follows: 

     vista_ncsc_run source_files -Wcxx,-tsv
vista_ncsc

If you will be running this wrapper script to compile your design in preparation for simulation using either dynamic or static linking, in the command line with which you invoke the script you must include NC's -cflags option to indicate that Vista's -tsv option is to be used, as follows: 

     vista_ncsc source_files -cflags -tsv

As an alternative to either of the above, you can uncomment the following line in the Vista .vistarc file: 

     ## VISTA_GLOBAL_COMPILATION_OPTIONS="-tsv"

For details on using Vista's TSV with simulation, see "Transaction Sequence Viewer". For information about using the .vistarc file, see the section "Controlling Vista Behavior".
