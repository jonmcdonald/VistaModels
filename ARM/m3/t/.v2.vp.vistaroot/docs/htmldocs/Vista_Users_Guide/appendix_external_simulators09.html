To create an ncsim executable that can be simulated from within Vista, instead of running a given tool using the original invocation command, you must invoke it via the appropriate Vista wrapper script.

Each wrapper accepts the same arguments as the corresponding Cadence tool that it invokes. (For information on using these arguments, see your NC-Sim documentation.)

The wrapper scripts also accept all the standard Vista command arguments. (For details, see t"Invoking Vista".)

All wrapper scripts are available in the /bin directory under the root of Vista's installation directory.
