Specifying PV Variables

PV variables that are defined here can be propagated to the timed model and viewed in analysis graphs. For more information see "PV Variables Propagated to T".

To specify PV variables for the model:

  1. In Vista's modeling dialog, select the PV tab.
  1. In the Variables tab, use the table to define one or more variables:
  • Name— Name of the variable.
  • Type— Data type of the variable.

Figure 7‑18. Vista Modeling - Variables Tab

  1. (Advanced) If you wish to change the kind of one or more variables from the default value, access them via the Attributes tab (see "Attribute Specifications for Variables and Registers").
  1. Click Save to save the settings.
