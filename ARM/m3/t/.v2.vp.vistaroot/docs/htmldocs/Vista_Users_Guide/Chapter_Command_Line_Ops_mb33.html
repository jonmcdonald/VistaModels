parse_regular_expression

Parses the protocol internal regular expression file or an external source file if specified.

Syntax

parse_regular_expression [-help] [-usage] protocol_path source_file

Arguments
Common (Optional)
  • -help
Provides help.
  • -usage
Displays the syntax for this command.
Required
  • protocol_path
Path of the protocol, for example, /TLM/protocols/ahb_master
  • source_file
Path to the regular expression external source file.
