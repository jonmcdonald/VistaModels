Simulate Learned Power Models With External Simulators

Vista Learned Power models can be simulated outside of Vista using other commercial simulators supporting the OSCI reference simulator (v2.2) and GNU tools (gcc) environment.

To enable simulation with an external simulator, you must have access to a Vista installation. In your make file include the following flags:

  • For gcc compilation:
     -DVISTA_POWER_EXT_MODEL -I vista_istallation/papoulis/include 
     -I vista_istallation/papoulis/include/mb_include
  • For linking:
     vista_istallation/papoulis/arch/libst.esl.library.so

Prior to executing the model outside of Vista, set the MGLS_HOME environment variable as follows: 

     setenv MGLS_HOME vista installation/papoulis/arch/mgls

Specify the architecture (arch) as is either linux32 or linux64.

Licensing

The execution of a model outside of Vista requires a vistavpeatsim license (P/N 101682).
