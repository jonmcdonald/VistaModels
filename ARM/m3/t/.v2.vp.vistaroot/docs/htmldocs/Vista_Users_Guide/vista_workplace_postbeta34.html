Vista Modeling Window

The Vista Modeling window is a modeling platform for engineers and system architects to efficiently create accurate TLM models that can be used for system-level validation or a reference virtual platform, and for performance/power trade-off analysis and optimization.

Following is a description of the menu commands that are accessible in Vista modeling windows. Wherever a tool bar icon and/or accelerator key or key combination can be used to invoke the same command, this is displayed in the table adjacent to the command name.

Table 3‑13. Modeling Commands and Toolbar Equivalents  

 

Command

Description

File

   

Save

Saves the model definition (.mb) file

 

Save As

Saves the model definition (.mb) file under a new name

Save & Generate

Saves the model definition (.mb) file and generates source files

 

Close

Closes the model builder dialog

Edit

   

Cut

Deletes the selected text and copies it to the clipboard

Copy

Copies selected text to the clipboard

Paste

Inserts text saved on the clipboard

Tools

   

Update Tables from Template

Opens the Update from Template dialog for generic models

 

Load Machine Architecture Script

Opens the file selection dialog for selecting a script to be used for populating the tables and field in the modeling tabs

Help

   

(F1)

Help on Current Tab

Opens the Help utility
