vista_simulate

This script performs batch simulation of the specified project.

Syntax

vista_simulate [-wd working-directory] [-params parameters-file] [-project project]
[[-design] design-specification [-- command-line-arguments]]

Arguments
  • wd working-directory
Specifies an existing working directory. (The default working directory is the one that was used when Vista was last invoked in the interactive mode after changes were made to the specified project.)
  • -params parameters-file
Specifies the path to a file containing parameters to be used during simulation.
  • [-project project]
Name of a Vista project.
  • [-design] design-specification
Designates which design in the project is to be simulated. This argument may be omitted if only one design exists in the project and no command-line arguments are specified. When designating a design-specification, it is sufficient to provide a sub-string that identifies the design uniquely among all the designs belonging to the project.
  • command-line-arguments
Any of those provided by your simulator.
