Modeling Objects Aggregation

In order to aggregate various modeling objects into compounds, use the modeling object mb_domain. The mb_domain class supports the following operations: 

     void mb_domain::add(mb_object* object);
     void mb_domain::remove(mb_object* object);
     void mb_domain::enable(void);
     void mb_domain::disable(void);
     void mb_domain::disable_and_reset(void);
     void mb_domain::set_enabled(bool value);
     bool mb_domain::is_enabled(void) const;
     void mb_domain::reset(void);

Except for add and remove, all other operations are propagated to the all domain members. Domains can be used for enable, disable, and reset of multiple modeling objects.
