Analysis Command Reference

The following table provides a brief description of each Vista Analysis command. Command details, arguments, and examples are available by selecting the links in the Command name column.

Table 18‑5. Machine Architecture Commands 

Command name

Description

vista_analysis

Opens the Analysis window for the specified simulation directory.

vista_dump_events

Creates a report of the events that occurred during simulation.

vista_summary_report

Creates a simulation summary report, throughput, latency, etc. for all simulation times.
