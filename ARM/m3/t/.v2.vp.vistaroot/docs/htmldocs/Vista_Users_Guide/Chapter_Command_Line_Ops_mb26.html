learn_power_hdl

Learns the power behavior by activity of HDL signals of the model.

Syntax

learn_power_hdl [-h | -help] [-usage] [-allow_file file-path] [-clustering_mode mode] library_name model_name

Arguments
Common
  • -h|-help
Provides help.
  • -usage
Displays the syntax for this command.
Optional
  • -allow_file file-path
Specifies path to file that lists the signal activity that is to be inputted to the learning engine. The default file path is path_to_model_rtl_power_folder/allowed_signals.txt.
  • -clustering_mode mode
Specifies clustering (maximum clusters and accuracy) to use. By default, 20,0.7 is specified and a maximum of 20 clusters with 0.7 accuracy is used. Specify no to avoid clustering.
Required
  • library_name
Specifies the name of model library.
  • model_name
Specifies the name of the model.
