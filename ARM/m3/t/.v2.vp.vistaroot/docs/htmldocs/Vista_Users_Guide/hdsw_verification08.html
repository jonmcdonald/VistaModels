Reset Commands

Core Reset

mon core reset

Resets the simulation core connected to the debugger.

After execution of the mon core reset command, the core is placed in the initial state. To continue software execution, a continue command must be issued.

Clicking the Core-Reset icon in the Sourcery CodeBench debugger control toolbar activates a monitor core reset command.

Reset

mon reset

An alias for mon core reset command.

Core Reset Request

mon core reset-request [RESET | NORESET]

Requests a system reset by asserting an external reset-request signal, if applicable. If NORESET is specified, the external reset-request signal is deasserted.

Clicking the System-Reset icon in the Sourcery CodeBench debugger control toolbar activates the mon core reset-request command.

The board designer is responsible for routing the reset-request to a reset controller that should reset all the devices including the current simulation core.

Reset Request

mon reset-request [RESET | NORESET]

This is an alias for mon core reset-request command.
