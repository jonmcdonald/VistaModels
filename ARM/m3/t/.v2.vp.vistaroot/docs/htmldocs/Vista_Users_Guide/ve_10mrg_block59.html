Creating a New Symbol

To create a new symbol for the current unit —

  1. In the Symbol Editor window, click the Create New Symbol icon: .

The pins representing the unit interface ports are displayed. In the Symbol list box, a new Untitled entry is listed.

Figure 9‑16. Untitled Entity

  1. Choose the Attributes > Page Setup command and, in the displayed dialog box, specify the orientation and page size to be used for this new symbol.

  Note

By default, the size of the page is user-defined. However, you can select any alternative standard size. Once a symbol has been assigned a specific size, it can be changed, as long as the symbol figures are not truncated as a result of decreasing the symbol size.

  1. Use the available drawing operations to create a geometric representation of the unit (see “Drawing in the Symbol Editor”).
  1. Move or rotate the pins (initially positioned at the edges of the window) as required, while drawing the symbol so that these pins are attached to the symbol's border.
  1. Save the symbol.

  Note

By default, the size of the page is user-defined. However, you can select any alternative standard size. Once a symbol has been assigned a specific size, it can be changed, as long as the symbol figures are not truncated as a result of decreasing the symbol size.
