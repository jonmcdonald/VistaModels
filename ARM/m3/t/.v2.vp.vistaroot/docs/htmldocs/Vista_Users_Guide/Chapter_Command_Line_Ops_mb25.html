learn_model

Learns an HDL model from a set of TLM databases.

Syntax

learn_model [-h | -help] [-usage] [-include-dirs dirs] [-progress-cmd cmd] [-rand-seed seed]
[-timing-model-source timing_model_source] [-use-timing-policies]
[-genetic-confidence confidence] [-lp | -learn-power] [-lt | -learn-timing] library_name model_name db_names

Arguments
Common
  • -help
Provides help.
  • -usage
Displays the syntax for this command.
Optional
  • -include-dirs dirs
Include directories for the timing model.
  • -progress-cmd cmd
Progress indicator command (for customizing the progress command).
  • -rand-seed seed
Random seed for learning; is used to guarantee identical learning results.
  • -timing-model-source timing_model_source
Source file of the timing model.
  • -use-timing-policies
Timing policies are used during learning (valid when specifying -learn_timing).
  • -genetic-confidence confidence
Optimization level, from 1 for short inaccurate learning to 8 for a model with highest accuracy.
  • -lp | -learn-power
Include power learning.
  • -lt | -learn-timing
Include timing learning.
Required
  • library_name
Name of model library.
  • model_name
Name of model
  • db_names
List of HDL simulation database names.
