set_tlm_library_physical_path

Sets the physical path for a library. This path will be the target for generation of future models that exist in this library.

Syntax

set_tlm_library_physical_path [-h | -help] [-usage] library_name physical_path

Arguments
Common (Optional)
  • [-h | -help]
Displays help.
  • [-usage]
Displays the syntax for this command.
Required
  • library_name
Name of the library
  • physical_path
Physical path of the library
