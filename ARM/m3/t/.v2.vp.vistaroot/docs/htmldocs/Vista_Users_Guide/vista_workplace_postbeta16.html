File Menu

Table 3‑1. File Menu Items 

Item

Description

Open Library Manager

Opens the Library Manager dialog for creating a new library or editing a library name or physical path

New File

Ctrl+N

Initiates creation of a new textual file by opening the New File dialog box.

New Model

Opens the modeling dialog for creating a new model

New RTL impl Model

Opens the modeling dialog for creating a new RTL implemented model.

New Schematic

Opens the Block Diagram editor (after prompting for a library)

Open

F2

Opens a file-selection dialog.

Close

Closes the file displayed in the current text window buffer.

Invoke External Editor

Opens the textual editor that you have designated as your external editor of preference. For details, see "Editing Source Files in Vista".”

Save

F3

Saves changes to the source file displayed in the current text window buffer.

Save As

Opens the Save As dialog to save the file displayed in the current text window buffer under a different name.

Save All

Saves all textual files that have been opened and modified.

Load Environment

Opens the Load Environment dialog to specify the initialization (vista.ini) file.

Print

Ctrl+P

Sends the contents of the source file displayed in the current text window buffer to the specified printer.

Exit

Ends your Vista session.
