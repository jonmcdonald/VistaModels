Notes and Limitations

  • Platform: Currently Linux is the only supported platform. Solaris and Windows(MingW) versions will be provided in a future release.
  • 32-bit mode: Currently Vista is not ported to 64 bit. MTI_VCO_MODE is set to 32 internally and this causes Questa to work in 32-bit mode.
  • Cdebug: In this release it is not possible to work with Vista and CDebug functionality at the same time, hence the Cdebug functionality is disabled. This will be changed in future releases of Vista.
  • Questa Version: 10.0a.
