remove_tlm_library

Removes a TLM library.

Syntax

remove_tlm_library [-h | -help] [-usage] library_name

Arguments
Common (Optional)
  • [-h | -help]
Provides help.
  • [-usage]
Displays the syntax for this command.
Required
  • library_name
Name of the library to be deleted.
