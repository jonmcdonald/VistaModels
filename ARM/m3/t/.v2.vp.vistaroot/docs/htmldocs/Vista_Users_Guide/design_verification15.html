Disabling Breakpoints

When you disable a breakpoint, it remains defined but will not influence the simulation run.

To disable one or more breakpoints:

  1. Select the breakpoint(s) in the Browser or in the source code.
  1. Choose the Simulation > Breakpoints > Disable Breakpoint command.

To disable all the breakpoints set for the design being simulated:

  • Choose the Simulation > Breakpoints > Disable All Breakpoints command.
Enabling Breakpoints

Having previously disabled one or more breakpoints, you can re-enable them.

To enable one or more breakpoints:

  1. Select the breakpoint(s) in the Browser or in the source code.
  1. Choose the Simulation > Breakpoints > Enable Breakpoint command.

To enable all the breakpoints set for the design being simulated:

  • Choose the Simulation > Breakpoints > Enable All Breakpoints command.
