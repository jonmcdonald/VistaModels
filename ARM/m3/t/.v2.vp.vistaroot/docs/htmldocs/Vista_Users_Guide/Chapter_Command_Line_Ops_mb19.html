generate_power_model_code

Creates a power model based on a state power policy defined in a TLM module.

The model contains all the variables involved in the state power policy. You must instantiate the model in your SystemC module.

Syntax

generate_power_model_code tlm_model_path [output_directory]

Arguments
Common (Optional)
  • -help
Provides help.
  • -usage
Displays the syntax for this command.
Optional
  • output_directory
Specifies the output directory for the tlm_model_name_power_model.cpp and .h generated files. If not specified, the model library directory is used.
Required
  • tlm_model_path
Specifies the logical path to the TLM model.
