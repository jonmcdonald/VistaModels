vista_rebuild_project

This script performs a batch rebuild of the specified project. It first deletes all the project's objects (.o files), then recompiles all its sources and builds the design.

Syntax

vista_rebuild_project [-wd working_directory] project_name

Arguments
  • -wd working_directory
Specifies an existing working directory. (The default working directory is the one that was used when Vista was last invoked in the interactive mode after changes were made to the specified project.)
  • project_name
Name of a Vista project.
