parse_verilog_source

Parses the Verilog source files into the model builder HDL with library_name.

Syntax

parse_verilog_source [-help] [-usage] [-directives directives] source_files library_name

Arguments
Common (Optional)
  • -help
Provides help.
  • -usage
Displays the syntax for this command.
Optional
  • -directives directives
Passes the desired compiler options to the compile command (depends on the Verilog compiler used).
Required
  • source_files
Verilog source file path.
  • library_name
Target HDL library name.
