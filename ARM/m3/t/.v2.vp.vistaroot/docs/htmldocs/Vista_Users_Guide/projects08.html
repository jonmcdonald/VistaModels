Defining Project Files

The Files tab is used to populate the various folders defined for a project with files already residing in your file system.


  Note

Always keep in mind that Vista does not copy or delete files from your system. Rather the display of files in the Vista's virtual folders is the result of referencing these files only. Thus, copying, moving, and deleting of files in the Vista folders is in fact copying, moving, and deleting the appropriate references.

Within this tab:

  1. Use the directory list box to browse to any location.
  1. Once you have reached the desired directory, select the file or files that are to be referenced by the specified folders. (You can select individual files or multiple files. To select multiple files, hold down the Shift key as you click on the name of the topmost and bottommost file in the list to be selected. To de-select individual files from the set of selected files, hold down the Ctrl key as you click on the name of the file to be de-selected.)
  1. Once you have selected one or more files, you have a number of possible ways of adding them to the appropriate folder.
  • If you have previously associated file extensions with the different folders ("Defining Project Folders") you can now utilize the Add by Extension option at the bottom of the dialog box.
  • Alternatively, you can select the Add to Folder control, and then specify the name of the folder that is selectable from the drop-down list.
  • You have the additional option of specifying the files to be stored in the appropriate folders using either a path variable or a full path. The drop box to the right of the Create button displays the names of any existing path variables. You can define a new path variable by clicking the Create button, and then specifying the name of the variable and the path with which it is to be associated.
  1. Click the Add button to create the link between the selected file(s) and appropriate folder(s).
