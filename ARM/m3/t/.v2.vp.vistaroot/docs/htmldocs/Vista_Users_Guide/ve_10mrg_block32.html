Creating Hierarchical Block Diagrams

Vista block diagram allows defining a hierarchical design. It is possible to instantiate block diagrams' design units within another block diagram. Vista allows creating multiple design hierarchies in a Bottom-up mode, that is, you need to create the lower level block diagram first, then instantiate it in a new upper level block diagram. To make such a design meaningful, the lower level block diagram should include external sockets (connectors) that bind to the upper level sockets in hierarchical-binding.
