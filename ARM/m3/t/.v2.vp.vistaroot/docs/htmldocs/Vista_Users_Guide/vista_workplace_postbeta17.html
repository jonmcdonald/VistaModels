Edit Menu

Table 3‑2. Edit Menu Items 

Item

Description

Undo

Ctrl+Z

Undoes the last editing operation or operations in the current text window buffer, depending on the number of times you invoke the command.

Redo

Redoes the last undone editing operation or operations in the current text window buffer, depending on the number of times you invoke the command.

Cut

Ctrl+X

Cuts the selected text from the editing window and places it onto the clipboard. In addition, this command applies to files, projects, and folders displayed in the Browser.

Copy

Ctrl+C

Copies the text selected in the editing window and places it onto the clipboard. In addition, this command applies to files, projects, and folders displayed in the Browser.

Paste

Ctrl+V

Pastes the contents of the clipboard into the textual window, starting from the position of the cursor (textual insertion marker). In addition, this command applies to files, projects, and folders displayed in the Browser.

Delete

Del

Deletes the selected text from the editing window. In addition, this command applies to files, projects, and folders displayed in the Browser.

Select All

Ctrl+A

Selects all the text in the source file currently displayed in the text window.

Find

Ctrl+F

Searches for the entered textual string.

Find Next

Searches for the next occurrence of the same string.

Replace

Searches for the entered textual string and replaces it with the one you specify.

Go to Line

Moves the cursor (textual insertion marker) to the line of text you designate in the opened dialog box.
