Instantiating a Component

To instantiate a unit as a component in a block diagram, do one of the following:

  • Select a model from a library, copy and paste it in the block diagram (use RMB menu for both).
  • Use the Get Component command from the block diagram.
  1. Click the Get Component icon

In the editor window, a pane appears that displays:

  • the libraries in your environment
  • the units in the selected library
  • the component symbols for the selected unit
  • a picture of the selected symbol with a Get button below the picture.
  1. Select the library, unit, and symbol that you want. (Every component has a default symbol. However, you can also define one or more of your own symbols for a given component. For details, see Instantiated Components.)
  1. Click the Get button.

An outline of the component instance moves with your mouse.

  1. Click where you want to position the component instance. (Like any graphic element, the component can be moved later on.)
  1. Connect the component's pins to the appropriate pins of other components.
