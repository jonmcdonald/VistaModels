Defining Channel Contents

The role of any channel in a block diagram is to carry information from one point to another. As has already been indicated, in Vista, the information carried by channels is typed.

In Vista there are two default types (one scalar and one vector) that are in effect as you create designs.

Each channel drawn in a block diagram is initially assigned a default type. You can modify this default type in the Tools > Options > Default Types dialog box.

The Default Scalar Type that you define is used as the default type throughout Vista.

For any channel drawn in a block diagram, you can override this default type. To do so —

  1. Select the channel, and then click Attributes > Element.
  1. From the Type drop-down list, select the type.
