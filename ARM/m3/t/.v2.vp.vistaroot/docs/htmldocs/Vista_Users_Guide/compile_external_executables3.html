Example

To illustrate, if your makefile uses the variables CC and CCC, you can either modify the makefile itself or invoke it as follows: 

           make CC=<vista_location>/vista_cc CCC=<vista_location>/vista_c++

Specifying the vista_cc and vista_c++ compiler wrappers as in the above example causes the default version of each compiler to be used. For details on setting the defaults, see "Controlling Vista Behavior", and "Invoking Vista".
