Running Simulation and Debugging

When you complete any of the three procedures described above, Vista will be invoked, and the Simulate dialog box displayed with the name of the appropriate executable appearing in the Target field of the General Tab.

To run simulation:

  • Click the OK button.

You will now enter Vista's simulation environment. Within the message pane the “ncsim” prompt will be displayed. There, you can enter any of the ncsim commands, as well as gdb commands. To do so, either:

At the ncsim prompt enter “gdb” followed by the desired command; for example 

     ncsim> gdb info breakpoints

Or

  1. At the ncsim prompt type “gdb”.
  1. Press the Enter key.

The prompt will now switch to gdb. Now you can directly enter any gdb command.

To return to the ncsim debugger:

  1. At the gdb prompt type “ncsim”.
  1. Press the Enter key.
