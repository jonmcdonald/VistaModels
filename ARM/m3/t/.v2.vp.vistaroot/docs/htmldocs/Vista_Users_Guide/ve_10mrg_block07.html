Structure & Flexibility of the Block Diagram and Its Generated Code

One of the capabilities of the Block Diagram Editor is to automatically generate code upon each save operation. The generated C++ file contains a SystemC module which has the same name as the Block Diagram. The generated code does not removes a previously-created file; this means that the user can modify the block diagram either by changing the graphics and saving the schematic, or by directly editing the source file.

To insure that the automatically-generated portions of the code do not conflict with code editing done directly by the user, a special methodology and syntax is used.
