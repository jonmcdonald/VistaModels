Fonts

Figure 9‑20. Block Diagram Options - Fonts Pane

The Fonts pane enables you to define the default settings for text appearing in different contexts:

  • Graphical element names in the Block Diagram graphical editor
  • Free text entered for documentation in these same graphical editors

When you click on the ellipses (...) button next to the appropriate context, a Select Font dialog box opens. Using this dialog box you set the default properties (font family, style, size, and color) for the fonts appearing in all subsequently defined units. While experimenting with various combinations properties, the result is displayed in the Sample window.
