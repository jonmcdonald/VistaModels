Instantiating Additional sc_modules in the sc_main

To instantiate additional sc_modules in a given sc_main:

  1. In the browser, select the sc_module classes of one or more header files.
  1. Choose the Add sc_module/s command from the right mouse button menu or from the drop-down menu accessed from the arrow located beneath the Tools menu. The Select SC_MAIN dialog box is now displayed.
  1. From the drop-down list, select the appropriate sc_main and click OK.

All instances that you selected in the browser are now added to the specified sc_main.

Note that it is your responsibility to define signals and connect the ports of the instances to one another.
