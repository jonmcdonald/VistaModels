View Menu

Table 3‑3. View Menu Items 

Item

Description

Show Browser Pane

Toggles the display of the Browser pane on or off. When the Browser pane is not displayed, the text window extends leftward.

Show Text Pane

Toggles the display of the Text pane on or off. When the text pane is not displayed, the Browser pane extends rightward.

Show Message Pane

Toggles the display of the Message pane on or off. When the Message pane is not displayed, the Browser pane and/or Text pane extends downward.

Show Options Pane

Opens the Options pane in the text-editor pane

Toggle XEmacs

Show/hide the menu for XEmacs editing.
