Vista Batch Scripts

Vista supports performing several tasks in the batch mode. For each of these tasks, a specific script invokes batch execution.

These scripts are stored in the root of the Vista installation directory and are named using the convention vista_operation.

The available scripts, arranged alphabetically, are described below.

  • vista_add_files_to_project
  • vista_build_project
  • vista_build_tree
  • vista_create_project
  • vista_generate_library
  • vista_rebuild_project
  • vista_rebuild_tree
  • vista_simulate
  • vista_memory_printer
