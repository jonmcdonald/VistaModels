Pre-Built Packages Provided with Vista

Vista's installation contains the following pre-built packages:

Table 5‑1. Pre-Built Installation Packages 

Package name

Description

model_builder

Set of packages enabling the verification of PV/PVT models which rely on Vista's modeling technology.

mb_compatibility

For compatibility of older versions (3.1.1 and older) of Vista Model-Builder code

models_catalogue

Vista TLM2.0 models

tlm

TLM 2.0.1

hce

host code execution package

SCV

SystemC verification library

sdParameters

Mentor Graphics' SystemC Parameters API

communication_channels

Mentor Graphics' Communication Channels package

vista_tlm_tracer

Allows for automatic tracing

scams

SystemC AMS

catapult

Supports CatapultC include files.


The corresponding Vista package description files reside in $VISTA_ROOT/vista_packages and may be used as samples for creating additional package descriptions.

Package Prerequisites

In order to able to use these packages, you must select the appropriate check box for each of these packages in the Project > Project Settings > Link tab dialog box.

In order to able to use the SA packages:

  • Mentor Graphics's System Architect product must be installed on your system.
  • You need to set the SA_HOME environment variable to the root of your System Architect installation.
