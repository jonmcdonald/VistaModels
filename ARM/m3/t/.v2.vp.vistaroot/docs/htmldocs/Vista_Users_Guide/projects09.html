Defining Include Paths

Project settings can specify “include paths” to directories containing external header files used by the current project. Using the -I option, these paths are added to the compilation command line for each of the current project sources.

  1. From within Vista, use the Include Paths tab of the Project Settings dialog box to define include paths:
  1. Click the browse button (...) to the right of any line.
  1. Using the Select Directory dialog box now displayed, navigate to a desired directory and click OK. The specified directory will be added to the Include Path list.
  1. Carry on defining additional directories as required.
