Data Types Defined in the Vista Libraries

The following types have been defined in the SystemC specifications prepared by OSCI.

Table 1‑1. SystemC Specifications

Type

Description

sc_bit

2-value single bit type (similar to the VHDL type bit)

sc_logic

4-value single bit type (similar to the Verilog type base)

sc_int<n>

1 to 64 bit signed integer type

sc_uint<n>

1 to 64 bit unsigned integer type

sc_bigint<n>

arbitrary sized, signed integer type

sc_biguint<n>

arbitrary sized, unsigned integer type

sc_bv<n>

arbitrary sized, 2-value vector type (similar to bit_vector)

sc_lv<n>

arbitrary sized 4-value (1, 0, x, z) vector-type (similar to logic_vector)
