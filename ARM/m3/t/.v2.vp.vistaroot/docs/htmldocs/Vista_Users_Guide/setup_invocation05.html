Initialization and Project Files

An initialization (vista.ini) file is created in the working directory from which you invoke Vista. This file specifies:

  • All of the project files defined in your Vista work environment
  • Various operational settings ranging from the base time unit for simulation to the colors used for various data types in the waveform display

Each project (*.v2p) file designates design source files as well as settings specific to this project.

You should refrain from modifying the contents of the initialization and project files outside of the Vista environment (i.e., using your OS access to these directories and files).
