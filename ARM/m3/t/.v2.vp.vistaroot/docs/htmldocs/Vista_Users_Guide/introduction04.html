TLM2.0 Socket Initiators and Targets

Vista also supports tlm sockets types socketInitiator and socketTarget, using base, blocking and non-blocking transport types.

The TLM-2 transport interfaces pass transactions between initiators modules (which create new transaction objects and pass them on by calling a method of one of the core interfaces) and target modules (the final destinations for transactions).

Predefined TLM2.0 socket initiator types include:

  • tlm::tlm_initiator_socket
  • tlm_utils::simple_initiator_socket
  • tlm_utils::simple_initiator_socket_tagged
  • tlm_utils::multi_passthrough_initiator_socket
  • tlm_utils::multi_passthrough_initiator_socket_tagged
  • types derived from the above.

Predefined TLM2.0 socket target types include:

  • tlm::tlm_target_socket
  • tlm_utils::simple_target_socket
  • tlm_utils::simple_target_socket_tagged
  • tlm_utils::passthrough_target_socket
  • tlm_utils::passthrough_target_socket_tagged
  • tlm_utils::multi_passthrough_target_socket
  • types derived from the above.

Use of these types should be in accordance with TLM2.0 rules.

Most of these socket types require SystemC version 2.2.0 or higher; See the TLM2.0 documentation.
