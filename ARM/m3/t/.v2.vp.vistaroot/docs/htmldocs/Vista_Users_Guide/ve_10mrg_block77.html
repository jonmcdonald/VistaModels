Default Names

Figure 9‑24. Block Diagram Options - Default Names Pane

The Default NAmes pane specifies default names for each of the graphic elements of a block diagram.

  • Prefix/Suffix Text Boxes

Enter a prefix and/or suffix to be appended to the default name for each listed element. When you create an element, a running integer will be appended to the prefix.

  • Prefix/Suffix of Signals Connected to Component Pins

The prefix and/or suffix strings you enter in these boxes will be added to the names of signals connected to these pins. Note that this applies both to signals connected manually as well as to stubs that are connected automatically to the pins using the Add Stubs command.
