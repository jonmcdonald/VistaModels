declare_fifo_register

Declares a register of type FIFO (See "FIFO Registers".)

Syntax

declare_fifo_register [-h | -help] [-usage] [{-fifo_size | -f }fifo_size] slave_port name address width

Arguments
Common (Optional)
  • [-h | -help]
Provides help.
  • [-usage]
Displays the syntax for this command.
Optional
  • {-fifo_size | -f }fifo_size
FIFO buffer size, default is 16 bytes.
Required
  • slave_port
Access port(s) name(s). For more than one port, use curly brackets; {port1 port2}
  • name
FIFO register name
  • address
Address for the register
  • width
Register width in bits
