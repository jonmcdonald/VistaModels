generate_testbench

Extracts a Transaction sequence for selected ports from a TLM database. The list of transactions can be read and executed by a CPU model. This command can be used to automatically create a TLM testbench from the HDL simulation data.

Syntax

generate_testbench [-help] [-usage] [-tlm-types] library_name model_name transaction_ports simulation_db_name output_file

Arguments
Common (Optional)
  • -help
Provides help.
  • -usage
Displays the syntax for this command.
Optional
  • -tlm-types
Use types defined in the TLM interface.
Required
  • library_name
The name of the model library
  • model_name
The name of the model
  • transaction_ports
List of required port names to be extracted from the simulation database to the output file.
  • simulation_db_name
Simulation database object name (created by read_vcd).
  • output_file
Target output file path.
