ARM Cortex M3

Feature List
  • AHB Interface
  • ARM v7M architecture Instruction Set
  • Thumb Instruction set
  • Thumb2 Instruction set
  • MPU - Memory Protection Unit
  • NVIC - Nested Vector Interrupt Controller
  • System Tick Timer
Interface Ports
  • icode - Cortex M3 I-CODE AHB bus interface
  • dcode - Cortex M3 D-CODE AHB bus interface
  • system - Cortex M3 AHB system bus interface
  • ppb - Cortex M3, private peripheral bus external APB bus interface
  • nmi - Active high non mask able interrupt signal
  • reset_n - Active low reset signal
  • irq_<N> - Active high interrupt signal, up to 240 external interrupts lines
Generation Parameters (Generics)
  • IRQ_NUM — Number of implemented IRQs. The value of this parameter defines the value of the ICTR control register.
CPU Specific Parameters
  • systick_clock - SysTick embedded timer clock. The actual clock value used by the embedded timer is choosen according to the CLKSOURCE bit in SysTick CONTROL register, according to ARM documentation.
  • v7m_timing - Path to a file containing a table describing Cortex M3 V7m timing. By default, the Vista Cortex M3 timing policies file is used.
  • v7m_power - Path to a file containing a table describing Cortex M3 V7m power. Not used by default.
  • priority_bits - Number of priority bits implemented in the CPU. Possible values are 1 to 8. Default value is 8.
  • CPUID - Value of the CPUID register that is visible by the software. Default value is 0x412fc231.
  • PID0-PID7, CID0-CID3 - Identification registers. Default values are PID0=0, PID1=0xb0, PID2=0x1b, PID3=0, PID4=0, PID5=0, PID6=0, PID7=0, CID0=0x0d, CID1=0xe0, CID2=0x05, CID3=0xb1.
  • All common core parameters are available under “cpu.core.*”
