Project Settings

All definitions and settings made for a given project can be accessed by choosing the Project > Settings command. The displayed dialog box contains various tabs, each one being used for a specific type of definitions. These include:

  • General definitions
  • Path Variables
  • Folders
  • Files
  • Include Paths
  • Definitions
  • Link

These various tabs are described below in the section Creating a New Project. Note, however, that these tabs are accessible during your work session to enable you to modify existing definitions.
