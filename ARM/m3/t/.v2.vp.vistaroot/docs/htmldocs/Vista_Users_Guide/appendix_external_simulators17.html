Running the simv Build Programs in Batch

If you have a script that runs one or more of the above Synopsis programs, to invoke this script so that the resulting simv executable can be simulated in Vista, simply prepend the string “vista_run” before the name of your script.

For example, if the script that you have prepared is named buildsimv.sh, then you should invoke this script by entering the command:

vista_run buildsimv.sh
