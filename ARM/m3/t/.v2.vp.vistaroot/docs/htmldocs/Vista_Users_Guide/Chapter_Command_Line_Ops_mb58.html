set_register_value

Sets register initialization value. (See "Extracting Power from HDL Models".)

Syntax

set_register_value [-help] [-usage] register_name port_name transaction_name condition value

Arguments
Common (Optional)
  • -help
Provides help.
  • -usage
Displays the syntax for this command.
Required
  • register_name
Register name
  • port_name
Port name
  • transaction_name
Transaction name
  • condition
Condition for setting the value
  • value
Value
