vista_generate_library

This script performs a batch generation of graphical libraries from Vista project modules, so that they can be integrated in Visual Elite. (For more details, see the "Vista and Visual Elite".)

Syntax

vista_generate_library [-wd working_directory] project_name [-lib library_path]
[{-all_modules | -modules} module_1, module_2...module_n | -modules_file file_name]

Arguments
  • -wd working_directory
Specifies an existing working directory. (The default working directory is the one that was used when Vista was last invoked in the interactive mode after changes were made to the specified project.)
  • project_name
Name of a Vista project.
  • -lib library_path
Specifies the path of the Visual Elite graphical library to which the modules will be generated.
  • {-all_modules | -modules} module_1, module_2...module_n | -modules_file file_name

-all_modules — generates a graphical component for each of the modules belonging to the project.

-modules — specifies the path to a previously prepared text file that contains the names of the modules to be converted to graphical components.

-modules_file — list that contains all of the modules that you selected in step 1 of this procedure.
