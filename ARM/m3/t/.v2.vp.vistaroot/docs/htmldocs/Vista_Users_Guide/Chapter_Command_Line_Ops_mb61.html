set_nominal_clock

Sets the nominal clock value in MHz.

Syntax

set_nominal_clock [-h | -help] [-usage] value

Arguments
Common (Optional)
  • [-h | -help]
Provides help.
  • [-usage]
Displays the syntax for this command.
Required
  • value
Specifies the value of the nominal clock in MHz.
