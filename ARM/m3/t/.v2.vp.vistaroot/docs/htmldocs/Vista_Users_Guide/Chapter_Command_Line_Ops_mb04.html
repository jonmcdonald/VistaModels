Conditions

The following illustrates condition syntax (used in register assignments): 

                 size == 64
           (1 << HSIZE) == 4
                 (start_time > 1000) && (end_time < 100000)
           address == 0x204

Possible identifiers include:

  • bool has_been_error
  • unsigned long long currentTime
  • config::uint64 address
  • config::uint64 size
  • config::uint64 data
