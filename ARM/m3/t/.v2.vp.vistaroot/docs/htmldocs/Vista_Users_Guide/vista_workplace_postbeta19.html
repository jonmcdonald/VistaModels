Project Menu

Table 3‑4. Project Menu Items 

Item

Description

New Project

Initiates creation of a new Vista project by opening the Create New Project (General) dialog box.

Open Project

Opens a file-selection dialog to open an existing Vista project (.v2p file).

Add to Project

Allows adding one of the following to a project: New File, New Folder, or Files.

Settings

Ctrl+F8

Displays the Settings tabbed dialog box, thereby enabling you to view and modify the current settings for a project.

Import Makefile

Currently not implemented.

Export Project...

Currently not implemented.
