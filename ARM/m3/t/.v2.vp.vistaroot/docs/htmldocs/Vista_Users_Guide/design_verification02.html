The Simulation Run

Once a design has been successfully built, it is ready for simulation.

Simulation is performed to verify whether the behavior in the design is consistent with its original specification. If there is a discrepancy, then it is necessary to debug the design. (See "Invoking Simulation".)

Several simulation-result data sets can be saved for subsequent analysis. See also Restarting Simulation from the Beginning regarding transcripts and resimulation scripts.
