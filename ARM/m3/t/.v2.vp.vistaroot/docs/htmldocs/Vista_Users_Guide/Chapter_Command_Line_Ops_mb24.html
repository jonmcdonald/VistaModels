instantiate_template

Creates an instance of a template model (for example, a template model from Generic_Library).

Syntax

instantiate_template [-h | -help] [-usage] template_path model_name library_name

Arguments
Common (Optional)
  • [-h | -help]
Provides help.
  • [-usage]
Displays the syntax for this command.
Required
  • template_path
Source path for the template.
  • model_name
Name of the target model.
  • library_name
Name of the target library.
