mb_fifo_reg Class

The following summarizes mb_fifo_reg behavior:

  • An mb_fifo_reg is associated with a slave port. It behaves as a regular register except that it stores the incoming transactions for further processing:
  • For incoming Write transactions, it accumulates the information until its buffer is full, and only then is the Write callback function associated with it invoked.
  • For incoming Read transactions, it provides the information until the fifo is empty, and only then is the Read callback function associated with it invoked.
