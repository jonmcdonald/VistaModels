save_model

Saves a model.

Syntax

save_model [-h | -help] [-usage] [{-d | -directory} directory] [-g | -generate]
[{-vista-project | -v} vista_project] tlm_model_path

Arguments
Common (Optional)
  • [-h | -help]
Provides help.
  • [-usage]
Displays the syntax for this command.
Optional
  • [{-d | -directory} directory]
Target directory for the model file (.mb).
  • [-g | -generate]
A flag that controls whether or not to generate model files.
  • {-vista-project | -v} vista_project
Add to specified Vista project name.
Required
  • tlm_model_path
Model logical path (TLM/lib/model).
