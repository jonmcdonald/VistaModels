Browser Pane

The Vista Browser contains the Projects tab which displays all of your Vista Projects, and one or more Simulation tabs for specific Projects.

For each Simulation tab, three different views of your design environment are available, enabling you to see the structure of a design from various perspectives. These views are:

  • Full view
  • Design view
  • Code view

The main or general Vista Browser view is the Project view, which hierarchically displays all defined libraries and projects in your work environment. Each branch representing a library contains headers, sources and model files. Likewise, each project contains the C++ file structure, code structure (extracted from the source files), and designs that were built for the project. These various elements reside in the appropriate standard folders: Public and External Headers, Sources, Schematics (for block diagrams), Simulation, Others, and Documentation.

The Generic Library includes any licensed models, in two categories:

  • Generic versions of basic and peripheral components that you can customize: Bus, CPU, Cache, DMAC, INTC, Interconnect, Memory, Snoop and Timer
  • Cores, based on the QEMU processor emulator and include a timing and power layer that is provided by Vista: arm11mpX1, arm11mpX2, arm11mpX4, arm926, mips24k, mpc603e, ppc405.

  Note

This list is updated as new models and cores become available, independent of the Vista version number.


  Note

When you create a virtual prototype (see "Virtual Prototyping"), runtime licenses for all models are removed, allowing software engineers to have free usage privileges for Vista-based TLM platforms.

The protocols library includes various communication protocols that are common and widely used, such as ahb, axi, apb, usb, and generic processor. Protocols are assigned to TLM ports (sockets); this adds standard communication behavior to the model and streamlines model portability.

(Note) Users can create their own proprietary protocols textually and add them to the protocol library.

In Simulation mode, three different Browser views are accessed by clicking the appropriate icon. When you click one of the three icons in the Browser tool bar, the selected view will be displayed.

When you select an item in any Browser view, the contents associated with this item are displayed in the editing window.

In addition, the title bar of the main Vista Window displays what type of item is selected in the Browser pane. This label will be one of the following:

  • The word “Projects” when the Projects (Full) view is active
  • The name of a specific design or sub-design (preceded by the word “Design“) when the Design view is active
  • The name of a specific class when the Code view is active
