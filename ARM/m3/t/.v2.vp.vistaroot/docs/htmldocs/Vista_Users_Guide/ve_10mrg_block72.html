Page Setup

Figure 9‑19. Block Diagram Options - Page Setup Pane

The Page Setup pane enables you to specify one of a number of predefined or user-defined page configurations (orientation plus dimensions) to be used in block diagrams. The settings made in this panel define the default page size for each subsequent graphical design unit that will be created.

To define a user-defined page size, select “User_Defined” from the Paper size drop-down list, and then specify the desired dimensions in the adjacent width, height, and units fields.

Note that the global settings defined in this panel can be overridden locally by means of the Attributes > Page Setup command in the Block Diagram. When you locally change the values in the displayed dialog box, this modifies the page size of only the current unit. The global page size settings remain in force.
