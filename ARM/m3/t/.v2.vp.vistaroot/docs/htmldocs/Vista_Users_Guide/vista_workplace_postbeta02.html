Introduction

The Mentor Graphics Corporation Vista 3.0 product is, to date, the only ESL tool providing a unified design environment for modeling, assembly, verification and analysis of TLM2.0-based hardware platforms.

After a platform is verified and analyzed, Vista 3.0 generates a Virtual Platform which allows software engineers to run their software to verify that it functions properly and meets timing and power requirements.

This chapter describes the key steps in modeling, assembly, verification, analysis and virtual prototype creation for Vista TLM2.0-based designs.

To learn how to use Vista 3.0 so that you can design customized TLM2.0-based platforms, it is recommended to execute the Vista 3.0 tutorial example, provided with the tool.
