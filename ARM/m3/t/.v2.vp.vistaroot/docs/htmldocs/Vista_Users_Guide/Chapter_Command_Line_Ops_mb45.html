start_output_log

Enables dump of output messages to a ModelBuilder_output.log (papoulis_output.log) file.

By default, the output log is enabled.

Syntax

start_output_log [-h | -help] [-usage]

Arguments
Common (Optional)
  • [-h | -help]
Provides help.
  • [-usage]
Displays the syntax for this command.
See also

stop_output_log
