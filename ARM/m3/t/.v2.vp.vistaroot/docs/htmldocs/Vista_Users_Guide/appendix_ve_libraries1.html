Vista and Visual Elite

Vista can generate graphical symbols for use in the Mentor Graphics Visual Elite tool. In this way engineers can develop designs that integrate both HDL and SystemC, supporting graphics and/or text-based modeling while addressing the entire scope of system and hardware engineering and project management, focusing on SoC design methodologies.

This integrated approach is achieved by generating graphical component libraries compatible with Visual Elite from Vista modules. These libraries are automatically linked with Visual Elite's work space and browser: Converted modules can be freely instantiated as a component in a Visual Elite SystemC block diagram.
