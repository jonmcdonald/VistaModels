Defining Settings

First, you must define general settings for the generation of graphical libraries:

  1. Choose the Projects > Settings > General command.
  1. In the General settings tabbed dialog box now displayed, enter the name of a directory that will serve as the default repository for the graphical libraries that you generate from your Vista projects.
  1. Optionally select the checkbox labeled “Don't ask for graphical library path.” In this case, whenever you initiate the generation of such a graphical library, Vista will always write this library to the default directory you specified in the previous step.

Once you have specified these general settings, you can generate graphical libraries either interactively or in batch mode.
