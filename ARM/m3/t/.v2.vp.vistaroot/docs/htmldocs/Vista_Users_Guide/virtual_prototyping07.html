Creating VP-for-Windows from Vista GUI

To create a Virtual Prototype for Windows from Vista GUI, do the following:

  1. In the Vista Virtual Prototype dialog box, choose the Virtual Prototype for Windows option.
  1. In case an installer with “walk-through” agreement file is required, you may specify the file and its path under License File field.

It is up to you (Virtual Prototype creator) to decide about the EULA (End-User-License-Agreement) file content (The agreement file may be only the provider file, or the provider append to Mentor EULA file or anything else).


  Note

The user will be required to accept the license terms (EULA) only once - the first time he/she executes the VP executable.
