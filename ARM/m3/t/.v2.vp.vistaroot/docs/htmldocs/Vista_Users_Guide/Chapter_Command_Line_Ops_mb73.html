vista_build_tree

This script performs a batch build of the specified project, compiling only those sources and dependencies that require compilation, then links the resulting objects and builds an executable.

Syntax

vista_build_tree [-wd working_directory] project

Arguments
  • [-wd working_directory]
Specifies an existing working directory. (The default working directory is the one that was used when Vista was last invoked in the interactive mode after changes were made to the specified project).
  • project
Name of a Vista project.
