set_leakage_power

Sets leakage power. (See "Specifying Power Policies".)

Syntax

set_leakage_power [-h | -help] [-usage] leakage

Arguments
Common (Optional)
  • [-h | -help]
Provides help.
  • [-usage]
Displays the syntax for this command.
Required
  • leakage
Static power in mW
