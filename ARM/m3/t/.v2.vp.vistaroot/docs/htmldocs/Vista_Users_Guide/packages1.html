Packages

Vista Packages are independently-built code components accessible from Vista.

Such components may contain header files and compiled code (objects, archives, or shared libraries). In order to inform Vista about an installed Vista package, its description must be copied into one of the following pre-defined directory locations:

  • $VISTA_ROOT/vista_packages under the root of Vista installation
  • $HOME/.vista/vista_packages under your home directory.

When invoked, Vista first searches for package descriptions in the latter of these directories.
