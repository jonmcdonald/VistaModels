Using MATLAB TLM2.0 Functional Models in Vista

Beginning with release R2010a from MathWorks, you can create TLM2.0 models from a given Simulink design.

Once a Simulink model has been simulated and verified, you can generate SystemC functional models (TLM2.0 LT) of pre-defined algorithms. (You may refer to the “TLM Generation Introduction” section in the documentation of MATLAB & Simulink for detailed information).

When the models are created, you can reference them under the Vista model builder as “external TLM model” and instantiate them in a Vista design like any other model created in Vista. To do that, do the following:

  1. Open Vista Model Builder dialog box.
  1. Click on the Generate tab.
  1. Reference the model's header file and its class name. Refer to "Generating a Model"”.

  Note

You may want to define the same ports as defined in the external model's class, in Vista Model Builder dialog box.

As MATLAB TLM2.0 models are functional models, you may use timing and power policies of the Vista model builder to assign architecture attributes to the model.
