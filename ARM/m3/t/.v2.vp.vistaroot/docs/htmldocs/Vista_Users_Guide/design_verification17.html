Deleting Breakpoints

Deleting a breakpoint means removing it entirely from your design.

To delete one or more breakpoints:

  1. Select the breakpoint(s) in the Browser or in the source code.
  1. Choose the Simulation > Breakpoints > Delete Breakpoint command.

To delete all the breakpoints that have been set for the design being simulated:

  • Choose the Simulation > Breakpoints > Delete All Breakpoints command.
