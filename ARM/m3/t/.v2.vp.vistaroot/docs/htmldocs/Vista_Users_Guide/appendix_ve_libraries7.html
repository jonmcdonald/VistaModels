Exporting sc_module Parameter Information to Visual Elite

The sc_module parameters defined for a Vista project can be displayed in Visual Elite (in the Component > Attributes > Parameters dialog box). This requires that, within Vista, you first prepare a directives file that has the following format: 

     module <module_a_name>
     parameter <parameter_a1_name> <parameter_a1_type> <parameter_a1_default_value>
     parameter <parameter_a2_name> <parameter_a2_type> <parameter_a2_default_value>
     .
     .
     parameter <parameter_an_name> <parameter_an_type> <parameter_an_default_value>
     .
     .
     parameter <parameter_b1_name> <parameter_b1_type> <parameter_b1_default_value>
     parameter <parameter_b2_name> <parameter_b2_type> <parameter_b2_default_value>
     .
     .
     parameter <parameter_bn_name> <parameter_bn_type> <parameter_bn_default_value>

Having created a directives file, you specify its location in the Directives file field of the Tools > Generate Graphical Library dialog box. (See "Interactively Generating Graphical Libraries".)
