Modeling Menu

Table 3‑6. Modeling Menu Items 

Item

Description

Create New Model

Opens the model builder (New Model) dialog for defining and generating PV and PVT models.

New RTL impl Model

Opens the modeling dialog for creating a new RTL implemented model.

Delete

Deletes the selected model

Open

Opens the modeling dialog for updating and generating the selected models. If a protocol is selected, opens the protocol file.

Load Model File(s)

Uses the selected model-definition (.mb) file to recreate or update the model object.

Generate Code

Generates headers and source files (.h, .cpp) from the selected model(s)

Add to Project

Opens the Add Model Files to Project dialog, which allows you to add generated model files to one or more selected Vista projects.

Read VCD

Opens Read VCD dialog box.

Learn Model

Opens Learn dialog box.

Get most active signals

Opens the “Most active signals” dialog box.

Learn Power HDL

Opens the “Learn power HDL” dialog box.

Compile Protocol

Compiles the selected protocol source file

Show State Machine

Shows the selected protocol in a state-machine diagram


Tools Menu

Table 3‑7. Tools Menu Items 

Item

Description

New sc_main

Automatically creates an sc_main method template and instantiates the selected module class(es).

Add Clock

Opens the dialog for specifying a clock for the selected sc_main() source file.

Add sc_module/s to sc_main

Allows you to instantiate additional module classes in a given sc_main.

Generate Graphical Library

Generates graphical symbols compatible with Visual Elite libraries from a selected project.

Open Analysis


Opens the Analysis window for analyzing the current simulation results for the selected simulation tab.

Open Analysis Reports

Opens a dialog with Analysis Report options: full transaction report (transactions and/or attributes) or summary report.
