Opening the Block Diagram Editor

To open a window of the block diagram editor, do one of the following:

  • Create a new block diagram by clicking the New Schematic tool button, or clicking File > New > Block Diagram in the main bar (see above).
  • Open an existing block diagram by double-clicking its icon in the Browser.

The Block Diagram Editor window contains pull-down s, a graphical workspace, and a two-column icon palette. See "Block Diagram User Interface Elements" for details.
