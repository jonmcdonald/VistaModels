Feature List

Interface Ports
  • Initiator - AHB bus master port
  • n_irq - Active low interrupt signal
  • n_fiq - Active low fast interrupt signal
  • n_reset - Active low reset signal
  • itcm_acp — used to access I TCM from outside; protocol:generic, direction:slave, bitness:32
  • dtcm_acp — used to access D TCM from outside; protocol:generic, direction:slave, bitness:32
CPU Specific Parameters
  • dcache_size — L1 data cache size. Default is 32K. The parameter specifies the data cache size in bytes. The size must be an integer which is a power of two. You may configure the dcache_size to be of zero size in order to disable the data cache functionality.
  • icache_size — L1 instruction cache size. Default is 32K. The parameter specifies the instruction cache in bytes. The size must be an integer which is a power of two. You may configure the icache_size to be of zero size in order to disable the instruction cache functionality.
  • dtcm_size, itcm_size — Size of data/instruction TCM memories. The parameter specifies the tightly coupled memory size in bytes if one is applicable. The size must be an integer which is a power of two. You may configure the TCM memory to be of zero size in order to disable the TCM functionality. Only one of the following values is permitted: 0, 4K, 8K, … 512K, 1M (default is 8K).
  • initram (default: false) enable Instruction TCM on reset.
  • HAS_ITCM_ACP — Defines whether to create the itcm_acp port. The specified value must be a boolean. A 1 creates the itcm_acp port. The default is 0.
  • HAS_DTCM_ACP — Defines whether to create the dtcm_acp port. The specified value must be a boolean. A 1 creates the dtcm_acp port. The default is 0.
