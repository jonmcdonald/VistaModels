vista_analysis

Opens the Analysis window for the specified simulation directory. (See "Selecting a Simulation Database for Analysis")

Calling this command with no arguments causes the Analysis window to open with no simulation directory.

Syntax

vista_analysis [simulation_directory] [simulation_directory2] [-session session-path]

Arguments
Optional
  • simulation_directory
The simulation database path required to view analysis graphs.
  • simulation_directory2
Optional second simulation database path.
If a second path is specified, the Analysis window opens with the two simulation directories marked as A and B.
  • -session session-path
Opens the Analysis window with the specified session loaded.
