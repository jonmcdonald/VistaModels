set_clock_tree_power

Sets click tree power. (See "Specifying Power Policies".)

Syntax

set_clock_tree_power [-h | -help] [-usage] clock_tree

Arguments
Common (Optional)
  • [-h | -help]
Provides help.
  • [-usage]
Displays the syntax for this command.
Required
  • clock_tree
The average energy per clock cycle in mW
