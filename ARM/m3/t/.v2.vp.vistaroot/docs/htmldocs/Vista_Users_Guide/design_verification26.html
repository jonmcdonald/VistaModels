The Memory View

The Vista Memory View is similar to the Memory tab sheet of the Simulator Control window. It enables you to view multiple memory instances in a separate views and verify their content. To open a Memory View directly from the Simulation tool bar, click the Memory View icon (next to the Simulation Control icon). Specify the appropriate pointer value in the Start Address field of the Memory View. For information on locating the appropriate Address pointer value, see Locating Vista Memory Model Address Pointer.

Locating Vista Memory Model Address Pointer

To locate the address pointer for a particular Vista memory model, expand the PV > Fields folder of the model. The address pointer value is indicated after the field name as shown in Figure 12‑13. For example, the address pointer for the “ram1” model in Figure 12‑13 is “m_data_ptr”. To view the contents of the memory instance, you would specify “0xb05cc000” in the “Start Address” column of the memory view.
