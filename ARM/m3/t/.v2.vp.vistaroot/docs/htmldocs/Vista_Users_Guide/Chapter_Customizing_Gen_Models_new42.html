MPC 603e

Feature List
  • Generic Bus Interface
  • I-Cache
  • D-Cache
  • Interrupts
  • MMU
  • Advance Instruction Timing Control
Interface Ports
  • Initiator - generic bus master port
  • hreset - Active high hreset signal
  • sreset - Active high sreset signal
  • interrupt - Active high interrupt signal
CPU Specific Parameters
  • sysclk — A clock value used by embedded timer facilities (TB, DECT, PT)
  • arch32_timing — Path to a file containing a table describing MPC603E timing. By default, the Vista MPC603E timing policies file is used.
  • v7m_power — Path to a file containing a table describing MPC603E power. Not used by default.
