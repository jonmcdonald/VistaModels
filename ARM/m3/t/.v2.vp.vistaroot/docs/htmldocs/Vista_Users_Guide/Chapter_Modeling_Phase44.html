Configuring a Model from IP-XACT

Vista enables you to configure a predefined generic component with information extracted from an IP-XACT description generated by the ARM AMBA Designer tool. Currently only the NIC301 MB generic model is supported.

Appendix D, “IP-XACT Configuration of ARM Component”, explains how to create a TLM 2.0 model from an IPXACT RTL description and describes the types of model parameters supported, other models that are generated, as well as any other operations that can be performed on the model.
