extract_model_interface

Creates a Model Interface where all Ports and Parameters are automatically extracted from the HDL interface signals. You can automatically map an HDL unit interface into a TLM model interface. The HDL signals are mapped to TLM ports according to the regular expressions defined for each Protocol.

Syntax

extract_model_interface [-h | -help][-usage] hdl_unit_path library_name model_name

Arguments
Common (Optional)
  • [-h | -help]
Provides help.
  • [-usage]
Displays the syntax for this command.
Required
  • hdl_unit_path
Source HDL unit path.
  • library_name
Target TLM library name.
  • model_name
Target model name.
