Analysis

During TLM simulation, a transaction-level database is automatically generated. All TLM database components are compatible with one another, whether generated from VCD files or from TLM simulation.

The generated database contains all the traced transactions and power activity from the simulation. Using Vista's analysis tool, users may trace and analyze the transactions throughput, the latency of each transaction and the power consumption. The results generated by the analysis can be output as ASCII text reports or displayed via powerful 3D graphing functions: see "Analysis Graphs and Reports".

Users may test and analyze selected scenarios and compare deterministically multiple simulations with various configuration combinations (for example, bus layering, power distribution, test bus load, cache size, and data latency configurations).
