Resolved Channel Types

SystemC defines a number of resolution types which may be used when there is contention between two or more drivers on the same channel or port.

If you want to designate that a channel should be of the appropriate contention type, you should assign it the type “__resolved” in the Attributes dialog box. In the output generated for this unit, one of the following types will be used, depending on the role the channel plays:

  • sc_in_rv
  • sc_out_rv
  • sc_inout_rv
  • sc_signal_rv.
