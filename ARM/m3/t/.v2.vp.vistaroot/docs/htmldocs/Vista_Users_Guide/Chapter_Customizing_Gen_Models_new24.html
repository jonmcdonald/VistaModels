Processor Model Integration

All models have a PVT TLM2.0 wrapper and are displayed in the Vista browser under Generic_Library/Cores library.

Any model can be integrated into the Vista TLM2.0 platform:

  • It can be instantiated in a Vista graphical block-diagram (schematic).
  • You can review the model's header file in the Vista browser, and instantiate it textually in your SystemC top-level module.

The models support the following Vista simulation modes:

  • Loosely Timed (LT)
  • Approximately Timed (AT)
