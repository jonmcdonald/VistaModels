run_external_command

Executes shell commands in a Vista Power script or prompt.

Syntax

run_external_command [-h | -help] [-usage] [{-e | -env} env] [{-o | -output-path} output_path]
[{-r | -rand-seed} rand_seed] [{-s | -sources} sources] command

Arguments
Common (Optional)
  • [-h | -help]
Provides help.
  • [-usage]
Displays the syntax for this command.
Optional
  • [{-e | -env} env]
Environment variables (e.g. -env VISTA_SYSTEMC=1).
  • [{-o | -output-path} output_path]
Target output path.
  • [{-r | -rand-seed} rand_seed]
Select seed for random sequence.
  • [{-s | -sources} sources]
Optional list of source files (used internally for regressions).
Required
  • command
The command to be executed.
