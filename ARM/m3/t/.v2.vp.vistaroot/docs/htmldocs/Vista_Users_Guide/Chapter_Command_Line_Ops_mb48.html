add_delay_policy

Adds a 'delay' policy. (See "Delay Policy".)

Syntax

add_delay_policy [-h | -help] [-usage] [-d | -disabled] [{-l | -latency} latency] [{-s | -sync} sync]
[{-p | -power} power] port_or_transaction wait_states

Arguments
Common (Optional)
  • [-h | -help]
Provides help.
  • [-usage]
Displays the syntax for this command.
Optional
  • -d | -disabled
A flag for not using this policy
  • {-l | -latency} latency
Latency in clocks.
  • {-s | -sync} sync
Specifies that waits are to be used to synchronize PV and T.
  • {-p | -power} power
Specifies the power value in mW.
Required
  • port_or_transaction
Effect port or port.transaction
  • wait_states
Clock cycles of the port
