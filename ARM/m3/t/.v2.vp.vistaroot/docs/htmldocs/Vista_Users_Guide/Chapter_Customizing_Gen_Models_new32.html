ARM Cortex M0

Feature List
  • Reduced thumb/thumb2 V6-M instruction set
  • Up to 32 IRQ NVIC
  • Optional Systick timer
  • SCS system control space
  • Configurable Small/Fast MULS
Interface Ports
  • initiator - common initiator for all outstanding AHB-Lite transactions.
  • reset_n - active log power on reset.
  • nmi - non maskable interrupt input.
  • irq_0 ... irq_32 - up to 32 external NVIC controllable inputs
Generation Parameters (Generics)
  • IRQ_NUM: number of implemented IRQs.
CPU Specific Parameters
  • systick_support (boolean) - true/false. Defines whether the CPU has systick timer support.
  • systick_clock - Clock used for systick embedded timer (if present).
  • multiplier_type (string) - “fast”/”small”. Defines which kind of multiplier is implemented.
  • v6m_timing (string) - path to v6m timing policy tables.
  • v6m_power (string) - path to v6m power policy tables.
