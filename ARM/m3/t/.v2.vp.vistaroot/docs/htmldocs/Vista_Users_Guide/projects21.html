Changing Project Settings

To change any of the previously defined settings for project:

  1. Choose the Project > Settings command.
  1. In the tabbed dialog box now displayed, make the desired modifications in the appropriate tab.

Note that the settings that you can modify here are identical to those that were definable when the project was initially created. These include:

  • General definitions
  • Path Variables
  • Folders
  • Files
  • Include Paths
  • Compilation
  • Link

For a description of each of these, see the like-named section above, beneath “Creating a New Project.”
