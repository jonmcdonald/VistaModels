Setting the VP-Plus License

The VP-Plus license (Vista VP-Plus Ap SW PN 248471) is used for both Vista and Vista Virtual Prototype when simulating with Sourcery CodeBench. The VP-Plus license enables hardware simulation control (see Controlling Hardware Simulation) and enables you to access hardware platform peripherals registers and local variable views and set new values for them (see Accessing Hardware Objects).

You can switch the license feature ON (or OFF) using one of the following methods:

  • Set V2_VP_PLUS 1 environment variable:
        setenv V2_VP_PLUS 1

This can be done before starting simulation (co-debug) or the VP prototype.

  • Use run Virtual Prototype command with the following option: -vp-plus.

This is equivalent to setting the V2_VP_PLUS 1 environment variable.

  • Select the appropriate option in the Vista Simulation dialog box Software tab sheet.
  • Specify the vp-plus option during vista invocation (vista -vp-plus &).

This sets the appropriate option in the Vista Simulation dialog box Software tab sheet.

  • Specify the -vp_plus option when using the vista_simulate batch command.

Once the license feature is set, the license is checked during simulation. If an error occurs when checking the license, an error message is displayed and simulation is aborted.
