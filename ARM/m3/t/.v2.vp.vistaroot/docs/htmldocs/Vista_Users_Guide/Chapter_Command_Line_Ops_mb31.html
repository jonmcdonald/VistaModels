load_tcl_machine_arch

Reads a tcl script for timing/power learning.

Syntax

load_tcl_machine_arch [-h | -help] [-usage] tlm_model_path tcl_script_path

Arguments
Common (Optional)
  • [-h/-help]
Provides help.
  • [-usage]
Displays the syntax for this command.
Required
  • tlm_model_path
Path to the TLM model file.
  • tcl_script_path
Path to the TCL script.
