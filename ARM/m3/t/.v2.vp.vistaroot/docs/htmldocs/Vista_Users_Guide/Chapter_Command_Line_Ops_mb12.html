create_protocol

Creates a protocol in Vista model builder using a .tcl script.

Syntax

create_protocol protocol-tcl-script [-g | -generate-ipxact] [{-i | ipxact-dest-library} ipxact-dest-library] [{-v | -vendor} ipxact-vendor] [{-ver | -version} ipxact-version]

Arguments
Common (Optional)
  • [-help | -h]
Displays help.
  • [-usage]
Displays the syntax for this command.
Required
  • protocol-tcl-script
Path to the .tcl script file that defines the protocol.
Optional
  • -generate_ipxact
If present, causes generation of an IP-XACT Bus Definition named protocol.xml and an IP-XACT Abstraction Definition named protocol_tlm20.xml, describing the protocol. The definitions are at TLM 2.0 abstraction level, using transactional ports.
  • -ipxact_dest_library string
Defines the library name for the generated IP-XACT descriptions. If not present, the name Library is used. This option is only considered if the -generate_ipxact option is present.
  • -vendor string
Specifies the vendor name for the generated IP-XACT descriptions. If not present, the vendor name Mentor.com is used. This option is only considered if the -generate_ipxact option is present.
  • -version string
Specifies the version string for the generated IP-XACT descriptions. If not present, the version string 1.0 is used. This option is only considered if the -generate_ipxact option is present.
