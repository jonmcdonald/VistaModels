Adding a Clock to sc_main

To add a clock to an sc_main:

  1. In the browser, select a specific sc_main to which the clock specification is to be added.
  1. Choose the Add Clock command from the right mouse button menu or from the drop-down menu accessed from the arrow located beneath the Tools menu.
  1. In the New Clock dialog box now displayed, enter the appropriate clock specification (Clock Name, Period, Time Units, Duty Cycle, First Edge Time, and First Edge Value).
  1. Click OK. The new clock settings you just made are now added to the sc_main source file.
