Saving Units and Objects

Two commands are available for saving design units and objects —

  • File > Save. Saves the last changes made to the currently active unit or object.
  • Save As. Enables you to assign a name to a previously unnamed new unit or object, or make a copy of the currently active unit or object and assign the copy a new name. Note, however, that when using this command you can store the diagram only within the current library.

The Save Block Diagram As dialog contains the following fields:

  • Library — Identifies the name of the current library; cannot be changed.
  • Module (required) — Specify the new unit or object name.
  • IP-XACT Vendor — Specify the vendor identification as a string. The vendor identification is used during IP-XACT generation of the unit; the default value is Mentor.com.
  • IP-XACT Version - Specify a version identification as a string. The version identification is used during IP-XACT generation of the unit; the default value is 1.0.
