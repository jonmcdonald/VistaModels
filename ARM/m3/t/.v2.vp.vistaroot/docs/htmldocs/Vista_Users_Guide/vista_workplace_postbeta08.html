Code View

In the Design view, if you select an object and click Code View, you see the code structure of the selected object.

In simulation mode, a simulation tab is open (see "Simulation and Verification" for more information). In this mode, the Code view offers a comprehensive view of the C/C++ structure including class view, class derivation, methods and data members fully synchronized with the source files, thereby allowing you to understand the structure of the code (classes, fields, methods, functions and their attributes).

From within a Code view you can open the source code matching the selected code element in XEmacs.

The information shown for a Code view can include the following

  • Base Classes
  • Fields
  • Methods
  • Return Types

For details on how to configure the display of this information, see "Setting View Options in the Options Pane".
