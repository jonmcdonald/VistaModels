Protocol Attributes

General Protocol Attributes

(* address *):- Defines the protocol parameter that represents the address

(* block_size *):- The size of the date (1 byte, 2 bytes, 4 bytes, 8 bytes, …)

(* bus_master *):- This is a master protocol

(* bus_slave *):- This is a slave protocol

(* clock *):- The clock that controls the protocol; there must be one and only one

(* default_read *):- Default READ transaction

(* default_value *):- Protocol parameter default value

(* default_write *):- Default WRITE transaction

(* default_transition *):- This protocol transition is part of a default transaction

(* error *):- This transition represents an error in a transaction (data not transferred)

(* master *):- This protocol parameter/function is driven/issued by the master side

(* power_manager *):- Used only in the power protocol

(* protocol_error *):- Error-function definition

(* protocol_initial *):- Protocol State Machine initial state definition

(* protocol_SM *):- Protocol State Machine definition

(* slave *):- This protocol parameter/function is driven/issued by the slave side

(* transaction *):- Transaction function definition

Protocol Attributes (Multiple State Machines Only)

(* id *):- Identification parameter, for out-of-order transactions

(* SM_successor *):- Initial state of the next state machine

(* tlm_phase_start *):- tlm_phase to be issued at the TLM level at the entrance to a state machine

(* tlm_phase_end *):- tlm_phase to be issued at the TLM level at the exit from a state machine
