clean_tlm_library

Deletes library content.

Syntax

clean_tlm_library [-h | -help] [-usage] library_name

Arguments
Common (Optional)
  • [-help/-h]
Displays help.
  • [-usage]
Displays the syntax for this command.
Required
  • library_name
Name of the library to be cleaned.
