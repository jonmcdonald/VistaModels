Visibility

Figure 9‑22. Block Diagram Options - Visibility Pane

The Visibility pane enables you to specify which kinds of text accompanying graphic elements in a block diagram window will be visible. To make a kind of text visible, select its check box. To make that kind invisible, deselect the check box.

When you make these setting with this Options Manager panel, you define the default settings for each subsequent block diagram that will be created. Note, however, that you can override these default settings locally by choosing the Attribute >Visibility command in the Block Diagram Editor. When you use locally change the Visibility values using the displayed dialog box, this determines the visibility of any currently selected element, and all subsequently created elements of this type in the current unit.
