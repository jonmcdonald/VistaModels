3 - Declaration of the Protocol States

An enumeration variable should be declared that contains all the states defined in the state machine.

This variable should be initialized to the IDLE state. 

     
     typedef enum { IDLE, // Waiting for Chip Select
                    R_DATA, // Read Data
                    R_RESP, // Read Response
                    W_DATA, // Write Data
                    W_RESP  // Write Response
                            } PROTOCOL_STATES;
     
     PROTOCOL_STATES protocolState;
     
     (* protocol_initial *)
     initial protocolState = IDLE;
