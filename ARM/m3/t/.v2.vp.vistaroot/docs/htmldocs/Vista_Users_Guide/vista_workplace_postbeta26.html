Window Menu

The Window menu contains the commands listed below, as well as menu items for various windows representing Vista objects:

  • Simulation windows
  • Compilation windows
  • Waveform windows
  • Modeling windows
  • Block Diagram windows
  • Analysis windows

    Table 3‑8. Window Menu Items 

    Item

    Description

    Clear Messages

    Clears all messages currently displayed in the Message window.

    List All Buffers F6

    Lists the contents of the various buffers currently populating the XEmacs editing window.
