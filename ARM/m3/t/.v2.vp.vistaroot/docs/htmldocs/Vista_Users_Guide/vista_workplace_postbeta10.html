Setting View Options in the Options Pane

While the different available views provide detailed information about the design, often only a portion of this information is important. However, you can control the amount of information you wish to see at any given time, as well as to determine whether this information is expanded or grouped in a specific folder.

To display the View Options pane:

  1. Choose View > Options Pane, or click the Options Pane icon
  1. In the three-tabbed panel now displayed in the Text pane space, click the appropriate tab: Project View, Code View, or Design View.
  1. Select or deselect the checkboxes as appropriate to define what should be displayed in the current view. For the Code and Design views, specify whether each item should or should not be displayed in a separate folder.

The activity between the options you select or deselect and what is displayed in the Browser pane are synchronized, meaning that the Options Pane does not have to be okayed in order for the settings to take effect. When you close the Options Pane, the current settings are automatically saved.
