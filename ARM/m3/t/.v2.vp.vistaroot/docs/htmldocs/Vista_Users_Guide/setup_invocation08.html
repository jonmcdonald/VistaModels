Preventing Sourcing of .vistarc Files

To prevent Vista from sourcing the .vistarc file stored in Vista's installation directory, in the shell from which you will invoke Vista set the environment variable VISTA_NO_SYSTEM_RC to any value.

To prevent Vista from sourcing the file /home/$LOGNAME/.vistarc (where $LOGNAME is the user name for the local machine) in the shell from which you will invoke Vista set the environment variable VISTA_NO_USER_RC to any value.

To prevent Vista from sourcing either of the two above files (“a” or “b”), in the shell from which you will invoke Vista, set the environment variable VISTA_NO_RC to any value.
