Software Image Control Commands

Core Load

Prerequisite: In some cases, before using this command, you must execute a mon core reset command to disable any MMU/MPU settings on the core.

mon core load ELF-FILE

Loads specified Elf image file into the memory of the current CPU core using the CPU initiator sockets.

Loading is performed using TLM transport_dbg calls.

Load

mon load ELF-FILE

An alias for the mon core load command.
