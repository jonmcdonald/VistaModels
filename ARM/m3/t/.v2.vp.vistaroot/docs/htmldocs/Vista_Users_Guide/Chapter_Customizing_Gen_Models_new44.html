PPC440 (Based on PPC440x5)

Model Structure

Under cpu.PV

  • “core” : mb::models::ppc::core4xx — The code.
  • “dcache” : mb::models::cache — The data cache.
  • “icache” : mb::models::cache — The instruction cache.
Interface Ports
  • data_read_initiator — Data read master port.
  • data_write_initiator — Data write master port.
  • insn_read_initiator — Instruction fetch master port.
  • dcr_initiator — DCR initiator port. Address is DCRN*4.
  • int_irq — External interrupt input. Active high.
  • cint_irq — Critical interrupt input. Active high.
  • reset — Chip reset input. Active high.
CPU Specific Parameters

Parameters' paths shall be provided as Inst.PV.PATH.

  • dcache_size — L1 data cache size. Default is 32K.
  • icache_size — L1 instruction cache size. Default is 32K.
  • timer_clock — Clock used for embedded timers. If not set, core “clock” parameter is used.
  • pvr - Value of the PVR register.
  • fpu_support - Boolean value specifying whether the FPU support is enabled.
  • boot_tlb - A specification of initial identical (i.e virt=phys) TLB map used during ELF loading and ELF starting.

Default value: “0=0:4G:128M 63=0xfffff000:4K:4K”.

The string value of this parameter is parsed according to the following grammar:

           tlb_boot: (group WS)*;
           group: index '=' addr ':' size ':' page_size;
           index: DEC_INT;
           addr: DEC_INT |'0x' HEX_INT;
           size: DEC_INT ('K'|'k'|'M'|'m'|'G'|'g');
           page_size: DEC_INT ('K'|'k'|'M'|'m'|'G'|'g');

Where:

           WS: white spaces;
           DEC_INT: dec integer;
           HEX_INT: hex integer;

No white spaces are allowed in the middle of the group.

Each group is interpreted as a sequence of TLB entries starting from entry 'index'; from address 'address' to 'address' + 'size'. The range in mapped using 'page_size' pages, i.e 'size'/'page_size' TLB entries will be used.

  • All common core parameters are available under “core.*”.
  • All common cache parameters are available under “dcache.*” and “icache.*”.
