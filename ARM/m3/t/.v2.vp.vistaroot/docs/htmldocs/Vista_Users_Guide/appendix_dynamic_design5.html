module_vector

defined in: vista_schematics/module_vector.h

This class represents a vector of MODULEs. Like any other 'object_vector', 'module_vector' uses MODULE_CREATOR to build each MODULE object. module_vector class should be used as follows:

  1. Create a sub-class of `module_vector'.
  1. Define PORTs pointers to the original fields by using one of the following classes:

`sc_port_ref', `sc_export_ref', `initiator_socket_ref', `target_socket_ref'

  1. Instantiate `module_vector' in SCHEMATICS.

`sc_port_ref', `sc_export_ref', `initiator_socket_ref', `target_socket_ref' classes are defined internally in class `module_vector'. Each of this classes provides an access to the PORTs of the original MODULE.

ModuleVectorPrj, BusVectorPrj, BusPrj examples demonstrate how 'module_vector' class should be used.
