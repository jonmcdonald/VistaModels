Adding Elements to a Project

A number of different types of elements can be added to a project. These are described below.

Adding a New File

Say that you want to initiate the creation of a new textual file (a header file, source file, etc.) within the Vista editing window. To do so:

  1. Choose the Project > Add to Project > New File command. The New File dialog box is now displayed.
  1. If you want this file to be referenced by a given project, keep selected the Add to Project check box at the top of the dialog box. Then choose the project to which the file will belong from the drop-down list. (If you just want to create the file without associating it at this stage with a certain project, de-select the Add to Project check box.)
  1. Browse to the directory in which you want the new file to be saved.
  1. Enter the name of the new file and click OK.

An appropriate template will be displayed in the Vista editing window which you complete as necessary.

Adding a New Folder

To add a new folder to a project:

  • Choose the Project > Add to Project > NewFolder command. The Folders tab of the Project Settings dialog box is now displayed.
Adding Existing Files to a Project

To add one or more existing files to a project:

  • Choose the Project > Add to Project > Files command. The Files tab of the Project Settings dialog box is now displayed.
