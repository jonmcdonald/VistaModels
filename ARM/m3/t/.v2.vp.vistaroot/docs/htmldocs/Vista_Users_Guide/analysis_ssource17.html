Maximum/Minimum Analysis Values

Vista Analysis shows Maximum values, in addition to the average value. For Attributes both Maximum and Minimum values can be shown.

The Maximum value is the largest value within a specific time interval and the Minimum value is the smallest value within a time interval.

Maximum/Average/Minimum values are shown in the same color but with different line styles: Max values are shown as dashed lines, Average values are shown as solid lines, and Min values are shown as dotted lines.

Maximum values apply to most Vista Analysis types (Power Analysis, Throughput Analysis, Latency Analysis, Arbitration time, and Contention). Minimum values apply to any attribute analysis defined by you. Max and Min values do not apply to Cache Hit and Miss Ratios (non-informative) Bus Throughput and Power Distribution as both already include multiple results in the same view and more lines would obscure the results.

Maximum and Minimum values are applied through their respective checkboxes, located next to the current analysis type checkboxes.
