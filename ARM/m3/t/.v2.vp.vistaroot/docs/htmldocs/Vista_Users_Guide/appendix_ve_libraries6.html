Defining Additional Attributes for Generated Components

When you instantiate a Vista-sourced component in a SystemC block diagram, the Attributes dialog box for this component has the following tabs that enable you to make additional definitions as required:

  • Constructor Parameters. Enter here any values for constructor parameters. The parameter values must be entered in the order of the parameter definitions in the source constructor. These values must be delimited by commas between them.
  • Additional Includes. Enter here include references required to compile the module beyond those already defined
  • Template module parameters. If the module is a template, enter the values of the template parameters here. The parameter values must be entered in the order of the parameter definitions in the template class declaration of the source. These values must be delimited by commas between them.
