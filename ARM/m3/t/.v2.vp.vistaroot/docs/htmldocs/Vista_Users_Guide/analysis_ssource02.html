Analyzing Simulation Data

Vista's analysis tool provides the following features:

  • The design tree is displayed, allowing you to acquire objects to the analysis graphs conveniently
  • In the output graphs, the plots are color coded, and the simulation databases are differentiated by marker shape
  • User-defined variables can also be analyzed (“analog” value changes plotted across time)
  • Data is available in tabular format, allowing you to view objects sorted based on statistical values (for example, average power, throughput)
  • In the tree pane, a tabular Analysis Summary is shown for a simulation session, immediately to the right of the tree.
  • You can zoom into specific portions of the output graph, and the tabular report is updated accordingly
Tracing Transactions on Ports
  1. Select Simulation > Simulate/Resimulate. The Simulation dialog opens.
  1. Click the Tracing tab.
  1. If relevant, select the Trace all sockets checkbox in the TLM section of the Tracing tab sheet.
  1. Click OK.
