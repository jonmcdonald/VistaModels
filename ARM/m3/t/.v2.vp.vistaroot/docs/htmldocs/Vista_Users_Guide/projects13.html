Manipulating Libraries and Projects

For managing your design, Vista enables you to perform various editing operations on projects and libraries.

You can, copy, cut, paste, delete and drag & drop different units from libraries listed in the Vista browser. The possible operations are summarized below:

  • Copying/Cutting all kinds of units; models (.mb or .mifc) or block diagrams, from a library and pasting them to another library. For example:
  • Copying a physical file from Lib1:Model1 and pasting it to Lib2.
  • Cutting from Lib1:Model1 and pasting to Lib2.
  • Copying/Cutting all kinds of units; models (.mb or .mifc) or block diagrams, from a project and pasting them to a library. For example:
  • Copying a physical file from Lib1:Model1 referenced by the project, and pasting it to Lib2.
  • Cutting Lib1:Model1 reference object from a project and paste Model1 to Lib2.
  • Copying/Cutting all kinds of units; models (.mb or .mifc) or block diagrams, from a project and pasting them to another project. For example:
  • Copying a reference object from project1 to project2.
  • Cutting a reference object from project1 and pasting it to project2.
  • Copying all kinds of units; models (.mb or .mifc) or block diagrams, from a library and pasting them to a project; same as the “Add to project” command. This is to create a new reference in a project.

  Note

Attempting to cut a unit from a library and paste it in a project will generate an error message.

  • Copying/Cutting a symbol from one unit and pasting it to another unit in the same/another library.
  • Deleting a unit or a symbol from a library/project.
Examples
  1. In Vista browser, select a unit (.mb or .mifc model or block diagram), then select `copy' from the main menu or the RMB menu. Choose a target library, then click the RMB and select `paste' from the popup menu.

The component that was copied will be displayed in the target library.

  1. In Vista browser, select a unit (.mb or .mifc model or block diagram), then select `cut' from the main menu or the RMB menu. Select a target library, then click the RMB and select `paste' from the popup menu.

The component will be displayed in the target library and removed from the source library.


  Note

You can copy/cut multiple units and paste them to one library.
 

You can also “drag and drop” or “copy and paste” models and symbols from libraries, listed in the Vista browser, directly to block diagrams.

Examples
  1. In Vista browser, select one unit (.mb or .mifc model or block diagram), then drag the mouse with the left button pressed. A label `library name:unit_name' will be displayed. Then drop the mouse in the block diagram editing area. The component will be displayed. Move the mouse to the desired location and click with the LMB to paste it.
  1. The same can be done with a component symbol. In this case, the specific symbol of the unit will be pasted to the block diagram.

  Note

You can “copy and paste” or “drag and drop” one unit at a time.
 


  Note

You can disable the “drag and drop” functionality in Vista by setting the VISTA_DISABLE_DRAG_DROP environment variable to 1.
