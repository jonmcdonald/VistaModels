T Model Callbacks

The following T Model callbacks (similar to the ones used in the bus) can be used: 

     /* called when "params.m_phase" is sent through the "port_index" port */
     virtual void sent_phase_cb(unsigned port_index,
        esl::sc_sim::mb_cb_params& params) {}
     /* called when "params.m_phase" is received by the "port_index" port */
     virtual void received_phase_cb(unsigned port_index,
        esl::sc_sim::mb_cb_params& params) {}

The esl::sc_sim::mb_cb_params has an element of transaction_status type that is an enumeration with 3 values: IS_START, IS_END, IS_INTERMEDIATE. This enables the callback code to work only for the start and end of the transaction if needed. With extended phases, BEGIN_REQ and END_RESP might not necessarily be the start and end of the transaction.


  Note

The methods accept_protocol_* are declared as deprecated and will be removed.
