vista_memory_printer

A Vista external utility that enables you to associate files under a host with memory models and print the file content (memory).

Syntax

vista_memory_printer {-file filename | -simdir dirname -module module_name}
[-address address] [-size element_size] [-count element_count] [-radix radix]
[-outformat columns] [-arch processor_archeticture] [-objdumpargs arguments]

Arguments
  • --file filename
Specify the path to the file to print.
  • -simdir dirname
Specify the path to the simulation directory that contains the module to print.
  • -module module_name
Specify the full path and name of the simulation memory module to print.
  • -address address
Specify the starting address of the memory location to print.
  • -size element_size
Specify the size of the memory elements to print.
  • -count element_count
Specify the number of memory elements to print.
  • -radix radix
Specify the print format as string, hex, dec, oct, or assembler.

  Note

When specifying an assembler radix, the VISTA_MEMORY_PRINTER_OBJDUMP environment variable is mandatory.

  • -outformat columns
Specify the number of columns to print. Default is 8.
  • -arch processor_archeticture
Specify the processor architecture of the module to print.
  • -objdumpargs arguments
Specify any extra arguments to pass directly to objdump as quoted ("") strings.
