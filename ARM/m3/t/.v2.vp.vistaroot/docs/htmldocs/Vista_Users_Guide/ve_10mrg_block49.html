File

Table 9‑4. Block Diagram File Menu Items 

Item

Description

Save

Ctrl+S

Saves changes to the file displayed in the current window.

Save As

Saves the file displayed in the current window under a different name.

Print

Ctrl+P

Sends the displayed contents of the schematic to the specified printer.

Printer setup

Opens a dialog containing printer options

HTML Capture

Opens a dialog containing HTML capture options

Close

Closes the block diagram window.
