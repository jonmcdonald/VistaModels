Package structure (directories and files)

  • vista_schematics - base classes for PORT_VECTOR and MODULE_VECTOR support
  • vista_schematics/pseudo_sc.h - pseudo SystemC types definitions
  • vista_schematics/socket_vector.h - SOCKET_VECTOR support
  • vista_schematics/port_vector.h - PORT_VECTOR support
  • vista_schematics/module_vector.h - MODULE_VECTOR support
  • vista_schematics/object_vector.h - Array of objects
  • vista_schematics/utilities.h - Utilities

This version of Vista will provide this vista_schematics directory as part of Vista installation.

examples - directory with examples.
