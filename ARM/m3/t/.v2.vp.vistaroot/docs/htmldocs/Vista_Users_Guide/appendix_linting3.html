Using Actis AccurateC SystemC Linter with Vista

Users who have licenses for AccurateC, Actis SystemC source code analyzer, can run this linter from within Vista. Doing so enables you to perform:

  • C++ linting of SystemC code
  • Netlist analysis across entire designs
  • Synthesis rule checking

You can also define your own rules for project-specific methodology.

In order to be able to do run AccurateC from within Vista, you must specify the installation directory of this tool in your environment using either the:

  • VISTA_ACHOME environment variable that can be set within the .vistarc file, or the
  • ACHOME environment variable

When you have done so, the Validate command is appended to Vista's Build. A command icon for invoking the linter is also added to Vista's tool bar:

The linter can be run on individual sources selected in Vista's browser or on entire projects. When you invoke validation, the results are displayed in Vista's buffer window. Clicking on any of the warnings or errors listed there jumps will cause the code displayed in the associated code to be displayed in Vista's text window.
