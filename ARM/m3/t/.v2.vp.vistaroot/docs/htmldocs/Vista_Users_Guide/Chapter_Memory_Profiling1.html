Memory Profiling

Vista is able to profile dynamic memory allocations for each SystemC process and show those allocations that were not freed until the current simulation time. For each process, the information shown in the Vista Memory Profiling Window includes:

  • Total potential memory leak
  • Number of potential memory leaks
  • Maximum allocation

Both table format and plot histogram are available.

The major benefit of Vista Memory Profiling is that you immediately learn which processes, if any, are suspect regarding memory leakage. As a result, you can further investigate only the problematic processes, avoiding the need to run expensive and exhaustive procedures over the entire design.
