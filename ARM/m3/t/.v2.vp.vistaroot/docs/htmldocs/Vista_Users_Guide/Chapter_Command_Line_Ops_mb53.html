declare_memory

Declares a memory range. (See "Specifying Memory Ranges".)

Syntax

declare_memory [-h | -help] [-usage] [-i | -is_virtual] memory_name slave_port base_address size

Arguments
Common (Optional)
  • [-h | -help]
Provides help.
  • [-usage]
Displays the syntax for this command.
Optional
  • -i | -is_virtual
A flag if this range is a virtual memory.
Required
  • memory_name
Identifying name
  • slave_port
Access port(s) name(s). For more than one port, use curly brackets; {port1 port2}
  • base_address
Start of the memory block
  • size
Size of the memory block
