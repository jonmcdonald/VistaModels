In order to create a simv executable that can be simulated from within Vista, instead of running a given program using the original invocation command, you must invoke it via the appropriate Vista wrapper script.

Each wrapper accepts the same arguments as the corresponding Synopsis program that it invokes. (For information on using these arguments, see your VCS documentation.)

The wrapper scripts also accept all the standard Vista command arguments. (For details, see "Invoking Vista".)

All wrapper scripts are available in the /bin directory under the root of Vista's installation directory.
