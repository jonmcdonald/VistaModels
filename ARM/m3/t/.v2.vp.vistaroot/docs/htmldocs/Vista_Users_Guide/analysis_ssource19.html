Analysis Toolbar

The elements of the Analysis toolbar are described here.

Table 14‑1. Analysis Toolbar  

Button

Command

Description

Save

Saves the current Analysis session. (Red oval indicates unsaved changes.)

Duplicate window

Copies the content to a new window.

Show/hide analysis browser

Displays/Hides the Analysis Browser pane.

Show/hide graph

Displays/Hides the Graph pane.

Show/hide report

Displays/Hides the Report pane.

Refresh

Runs analysis on and displays the selected data.

Refreshes the results according to the current state of the simulation.

Add objects

Adds the selected objects to the graphs and report tables

Remove objects

Removes the selected objects from the graphs/tables.

Remove all objects

Removes all objects from the graphs/tables.

Zoom in

Redisplays the graph showing more detail in a narrower time interval.

Zoom out

Redisplays the graph showing less detail in a wider time interval.

Full zoom

Redisplays the graph showing the entire simulation interval.

Previous zoom

Redisplays the previous time segment.

Sync Views

Allows you to view specific analysis of specific data in the Vista waveform context and vice versa.

Scroll to 0

Scrolls graph to 0 time on the left

Scroll 20% left

Scrolls graph showing time segment shifted 20% of the whole segment to the left

Scroll 10% left

Scrolls graph showing time segment shifted 10% of the

whole segment to the left

Scroll 10% right

Scrolls graph showing time segment shifted 10% of the whole segment to the right

Scroll 20% right

Scrolls graph showing time segment shifted 20% of the whole segment to the right

Scroll to end

Scrolls graph to the end of simulation on the right
