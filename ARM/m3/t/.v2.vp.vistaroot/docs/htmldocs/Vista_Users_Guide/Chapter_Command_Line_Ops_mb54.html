declare_register

Declares a register (See "Specifying Registers".)

Syntax

declare_register [-help] [-usage] [-is-trigger] [-kind kind] [-rw-access rw_access]
[-size size] [-width width] [-type type] [-rv | -reset_value value] slave_port register_name register_name bit_range

Arguments
Common (Optional)
  • -help
Provides help.
  • -usage
Displays the syntax for this command.
Optional
  • -is-trigger
A flag if the register should have a callback method
  • -kind kind
Register kind, default is bitfield, can be also number or address
  • -rw-access rw_access
Read/Write access - values can be r, w, or r/w.
  • -size size
Size of the register.
This option is deprecated, use the -width option.
  • -width width
Width of the register in bits.
  • -type type
Register type.
  • -rv |-reset_value value
The value of the register after reset model action.
Required
  • slave_port
Access port(s) name(s). For more than one port, use curly brackets; {port1 port2}
  • register_name
Identifying name for the register
  • register_name
Register address
  • bit_range
Bit range in format: (max:low)
