Visibility

(Attribute > Visibility)

Use to enable and disable visibility of different kinds of text accompanying elements in the current unit of a block diagram window.

Select the type of text to be visible. To make that kind invisible, deselect it.

Values in this dialog box determine the visibility of any currently selected element, and all subsequently created elements of this type in the current unit. The settings in this dialog box override the default values made under Tools > Options > Visibility.

Fonts

To access the Font dialog box, select a text object then choose Attribute > Fonts or use the Font command button, whenever it appears in a dialog box.

The Font dialog box is used to locally change the settings for text displayed in different contexts. The settings in the Font dialog box override the default font settings defined using Tools > Options > Fonts.

The settings for text in two contexts can be changed:

  • Graphical element names in the Block Diagram editor
  • Free text entered for documentation in these same graphical editor

Click on the ellipses (...) button next to the appropriate context, to open the Select Font dialog box. In the Select Font dialog box, set the default properties (font family, style, size, and color) for the fonts displayed in all subsequently defined units. When changing various property combinations, the result is displayed in the Sample window.

If your computer is equipped to display Japanese fonts, you can select the desired language standard for free text in your graphical units:

  • iso8859-1 for Western characters, which makes available Modern, Roman, or Swiss fonts
  • jisx0208.1983 for Japanese characters, which makes available Mincho or Gothic fonts

Click OK in the Select Font dialog box to apply the selected font properties to the appropriate Fonts panel context field. The font name, style, and size are displayed in the selected color.

Colors

To access the Colors dialog box, select a graphical object then choose Attribute > Color.

Enables you to modify the color of the selected object. Fill and line colors are assigned separately. Select the ellipses button (…) to open the color table from which a color can be selected and displayed in the dialog.
