Interactively Generating Graphical Libraries

To initiate the interactive generation of a graphical library from a project:

  1. In the Browser, choose the project, or one or more individual modules stored in a project, for which you want to generate graphical libraries.
  1. Choose the Tools > Generate Graphical Library command.
  1. In the dialog box now displayed, make the appropriate settings as described below.
  • Graphical library path is where the library will be written to. However, if you previously activated the “Don't ask for graphical library path> option, this browse field will now be inaccessible.
  • Directives file specifies the location of the directives file that must be prepared in order to be able to display the sc_parameters defined for a project in Visual Elite. (See "Exporting sc_module Parameter Information to Visual Elite".)
  • Generate modules in one of three possible modes.
  • All modules. A graphical component will be generated for each of the modules belonging to the project.
  • Load module names from file. Having previously prepared a text file containing the names of the modules to be converted to graphical components, when you select this option you can now browse to the location of the file.
  • Select modules from list. This list contains all of the modules that you selected in step 1 of this procedure.
  1. Click OK.

What next happens depends on the status of the “Don't ask for graphical library path” checkbox in the Projects > Settings > General dialog box:

  • If the checkbox is not selected, in the dialog box now displayed the “Select modules from list” mode will be automatically activated and the modules that you selected in the browser's project tree will be selected in the list box.
  • If the checkbox is selected, the generate library dialog box will not be displayed at all. Rather, the graphical units will automatically be generated to the default directory that you designated in your general settings (see above).
