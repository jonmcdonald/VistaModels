declare_variable

Declares a PV variable. (See "Specifying PV Variables".)

Syntax

declare_variable [-h | -help] [-usage] [{-k | -kind} kind] [{-t | -type} type] name

Arguments
Common (Optional)
  • [-h | -help]
Provides help.
  • [-usage]
Displays the syntax for this command.
Optional
  • {-k | -kind} kind
Variable kind. Default is bitfield; can be also number or address.
  • {-t | -type} type
Variable data type. Default is int.
Required
  • name
Variable name
