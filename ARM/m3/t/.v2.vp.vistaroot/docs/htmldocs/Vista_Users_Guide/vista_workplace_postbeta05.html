Vista Window

The Vista Window is divided into three main regions or panes:

  • The browser pane in the upper-left portion of the window, which displays a Projects tab and one or more Simulation tabs with three views (Project view, Design view, or Code view). See Full View, Code View, and Design View for further details.
  • A text pane in the upper-right portion of the window, in which you edit your design files using XEmacs. See Text Pane for further details.
  • A message pane at the bottom of the window, which has several purposes:
  • In Project mode (when the Projects tab is open), there are two subtabs:
    Vista Console (primarily for modeling commands: see "The Vista Workplace") and Compile Output.
  • When the Simulation tab is open, this pane is the XEmacs mini-buffer, which you can use for entering emacs-specific commands (search, load, etc.)
    During simulation, you can enter Vista or gdb command-line commands here.

(For further information, see Message Pane).

The currently-active pane is indicated by the color of the banner above the pane.

While this three-pane configuration is the system default, you can change the relative size of the different panes by dragging the frame lines between panes. You can also hide any one or two of the panes using the toggle commands described below:

To hide (or redisplay) the Browser pane:

  • Choose the View > Show Browser Pane command or click the corresponding icon —

When you hide this pane, the Text pane, if opened, will extend to the left and occupy its space.

To hide (or redisplay) the Text pane:

  • Choose the View > Show Text Pane command or click the corresponding icon —

When you hide this pane, the Browser pane, if opened, will extend to the right and occupy its space.

To hide (or redisplay) the Message pane:

  • Choose the View > Show Message Pane command or click the corresponding icon —

When you hide this pane, the Browser and/or Text pane(s), if opened, will extend downward to occupy its space.

Note that each pane can also be hidden by clicking the close icon located to the extreme right end of the banner above the pane:

This icon is accessible on a pane as long as it is not the only pane that is open.
