load_model

Loads a model from a source .mb file to the Vista tree, or updates existing model data from the corresponding .mb file.

Syntax

load_model [-h | -help] [-usage] [{-d | -dry} dry] library model

Arguments
Common (Optional)
  • [-h | -help]
Provides help.
  • [-usage]
Displays the syntax for this command.
Optional
  • [-{d | -dry} dry]
Dry mode - does not create a model; just reads the model content and returns the model's path; the default is 0.
Required
  • library
Library name
  • model
Model name or .mb file
