Wrapper Scripts

In order to build your design to be simulated on the Questa simulator, you need to use the Vista equivalent wrapper script for one or both of the following tools supplied by Mentor:

Table C‑1. Vista Wrapper Scripts

Task

Mentor's tool

Vista's wrapper script

Compilation, Linking

sccom

vista_sccom

Simulation

vsim

vista_vsim


Each wrapper accepts the same arguments as the corresponding Questa tool that it invokes. (For information on using these arguments, see your Questa documentation.) The wrapper scripts also accept all the standard Vista command arguments. (For details, see t"Invoking Vista".)

All wrapper scripts are available in the /bin directory under the root of Vista's installation directory.
