Cadence NC Simulators

Vista supports the use of Cadence's simulators both for pure SystemC designs (NC‑SC) and for designs containing hardware description language code (NC‑Verilog and NC‑VHDL). This integration is implemented by a number of wrapper scripts, each of which launches the corresponding NC tool.

From within Vista, you run NC simulation using the graphical SimVision environment, invoking it in one of two ways:

  • You can enter the “simvision” command at the ncsim prompt, as usual.
  • Or, you can set the “-gui” option in the vista_ncsc_run script (see below).

  Note

Currently, breakpoints in your SystemC code can only be set in Vista, and not using SimVision's graphical interface.
