ARM-FM Processors Library

The ARM-FM environment (version 7.0.x - including MI extension) is required to use the ARM-FM Processor Library. ARM-FM is installed by sourcing the source_all.chs script.

You can instantiate an ARM-FM core from the GenericLibrary/FastModels template library, using the same procedures you would to create and instantiate other Vista models:

  • Open the model (Generic_Library > ARM_FastModels > CPU-Name).
  • Specify the model configuration.
  • Click Save & Generate to save the model configuration and generate the model code.

  Note

For ARM Fast Models, the Save & Generate command automatically initiates ARM LISA and runs the appropriate scripts and Make files to build the AFM libraries.
Due to SystemC simulation timing it is recommended you use ARM-FM MI (Multi-Instance).
