associate_hdl_unit_to_model

Associates an HDL unit and signals with a TLM model and Ports.

Syntax

associate_hdl_unit_to_model [-h | -help] [-usage] [‑port_association_list list]
[{-hu | -hdl_unit_name} name] library_name model_name

Arguments
Common (Optional)
  • [-help | -h]
Displays help.
  • [-usage]
Displays the syntax for this command.
Optional
  • ‑port_association_list list
List of protocol parameter and HDL signal pairs for each port. The list format is:
{port1 {param1 signal1 param2 signal2} port2 {…}…}
  • {-hu | -hdl_unit_name} name
Specify the name of the HDL Unit. If not specified, HDL Unit name is assumed equal to the TLM model name.
Required
  • library_name
Specify the name of the model library.
  • model_name
Specify the name of the TLM model.
