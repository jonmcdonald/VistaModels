Advanced Modeling Topics

Vista offers an additional modeling style in which users can create mixed PVT models where timing/power and functionality are defined manually. This style is recommended only for advanced users and for very specific tasks: Please consult with your Mentor support team before using this approach.

The TLM model is generally interfaced through slave ports (inputs) and master ports (outputs).

Vista provides:

  • callback functions for incoming transactions and end of outgoing transaction interfaces for linking with external TLM functional models
  • functions for initiating output transactions

Refer to the Vista Modeling Guide and Application Notes as well as the information in this section.
