dump_simulation_data

Dumps an ASCII list of all transactions in the TLM database into a file. If an instance is specified, only transactions of that instance are dumped.

Syntax

dump_simulation_data [-h | -help] [-usage] [{-i | -instance_path} instance_path] db_path output_file

Arguments
Common (Optional)
  • [-h | -help]
Provides help.
  • [-usage]
Displays the syntax for this command.
Optional
  • [{-i/-instance_path} instance_path]
Hierarchical instance path
Required
  • db_path
Simulation data object path
  • output_file
Path to the target output file
