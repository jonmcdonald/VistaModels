SystemC Support

Vista provides a method for capturing and simulating SystemC designs, allowing you to create models of software algorithms, hardware architectures, and interfaces of SoC (System-on-Chip) and system-level designs.

Using Vista, you can create a system-level model, quickly simulate to validate and optimize the design, explore various algorithms, and provide both hardware and software development teams with an executable specification of the system. An executable specification is essentially a C++ program that exhibits the same behavior as the system.

Vista supports several SystemC 2.2 versions; by default, version 2.2.

Most of the concepts and semantics introduced in SystemC 2.x.are supported. These include:

  • Channels
  • Interfaces
  • Events
  • New model of time
  • Dynamic sensitivity through the wait() method.
