Troubleshooting

Vista provides two scripts that you can use for troubleshooting:

  • vista_run can be used to invoke any executable from within the Vista environment, giving you all the same settings as in Vista. For example: 
           vista_run xemacs &
  • vista_env returns the value of the specified Vista environment variables. For example, to return the path of the directory in which Vista looks for its online documentation, enter the command:
           vista_env VISTA_HELP_ROOT &
