Classes

Most of the pre-defined (“mb_”) classes can be created via the graphical user interface (Build Model dialog), as specified below:

  • mb_register (created via the GUI in Registers; see "Specifying Registers")
  • mb_memory (created via the GUI in Memory Ranges: see "Specifying Memory Ranges")
  • mb_memory_range (created via the GUI in Memory Ranges: see "Specifying Memory Ranges")
  • mb_variable (PV and T) (created via the GUI in PV Variables: see "Specifying PV Variables")
  • mb_event
  • mb_fifo_reg
