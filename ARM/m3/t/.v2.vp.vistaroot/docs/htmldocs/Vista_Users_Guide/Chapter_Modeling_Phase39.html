Extracting Power from HDL Models

Vista provides a method for extracting power from HDL models and creating a power view that can be attached to SystemC models. Extraction is done through protocols code, which maps pin activity to transactions and machine learning algorithms that translate the inputs to a power view. The power view can be either Vista power policies or power models.
