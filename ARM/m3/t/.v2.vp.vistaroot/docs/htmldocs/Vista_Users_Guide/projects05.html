Project View in the Browser

One of the main advantages of Vista over any other SystemC design tools is its ability to depict the structure of the design from various perspectives.

The Project View allows you to expand the header files and reveal the C++ class structure defined for the design. This expansion is possible down to any level along the class inheritance chain. In addition, data members and function (method) members of the class are also explicitly iconized.

Figure 4‑4. Project View

Together with the Code View, the Project View allows you to expand and reveal the module instantiation structure or the hierarchical decomposition of the design which is the Design View. In fact, Vista Project View allows you to simultaneously view both views relating to the same piece of C++ code.

Vista allows you to instantly select a class object declaration statement, object implementation code, or an object instantiation statement.

To do so:

  1. Select the icon for the appropriate item in the browser pane.
  1. Holding down the right mouse button, choose the applicable command:
  • Go to Declaration. Select the object in the Project, Design, or Code View, and then choose the Go to Declaration command. The text editor (for example, Xemacs) will highlight the line were this object is declared.
  • Go to Implementation. Select the object in the Project, Design, or Code View, and then choose the Go to Implementation command. The implementation of the object class will be highlighted, and its header file displayed, the beginning of the class definition header being selected.
  • Go to Instantiation. Select the object in the Project or Design view, and then choose the Go to Instantiation command. The instantiation of the object will be highlighted.
