put_causality_ban

Enables the removal of an incorrect causality between an input and output event.

Syntax

put_causality_ban [-help] [-usage] tlm_model_path port_name event_name

Arguments
Common (Optional)
  • -help
Provides help.
  • -usage
Displays the syntax for this command.
Required
  • tlm_model_path
Path to the TLM model
  • port_name
Name of the port to which the ban is applied
  • event_name
Name of the banned event
