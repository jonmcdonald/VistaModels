add_generated_files_to_vista_project

Adds model generated files to a Vista project.

Syntax

add_generated_files_to_vista_project [-h | -help] [-usage] tlm_model_path vista_project

Arguments
Common (Optional)
  • [-help/-h]
Displays help.
  • [-usage]
Displays the syntax for this command.
Required
  • tlm_model_path
Path to the source model.
  • vista_project
Name of the target Vista project.
