state_power

Adds power based policy. Refer to "State Based Power Policy"

Syntax

state_power [-h | -help] [-usage] [-d | -disabled] [{-condition_pairs | -c} condition-pairs] power

Arguments
Common (Optional)
  • [-h | -help]
Provides help.
  • [-usage]
Displays the syntax for this command.
Optional
  • -d | -disabled
A flag for not using this policy.
  • {-condition_pairs | -c} condition-pairs
List of “variable-names variable-values”, for example, “var1 10 var2 20 var3 10”
Required
  • power
Specifies power value in mW.
