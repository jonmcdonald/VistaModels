vista_add_files_to_project

Prerequisite: A Vista project file or name must exist before using this script.

This script add files to a Vista project in batch mode.

Syntax

vista_add_files_to_project [-wd working-directory] project_file_or_name {file1 file2 ... }";

Arguments
  • -wd working-directory
Specify an existing working directory.
By default, the working directory is the one that was used when the Vista product was last invoked in the interactive mode, after changes were made to the specified project.
  • project_file_or_name
Specify a Vista project filename or project name.
  • file1 file2 ...
Specify the name(s) of the file(s) to add to the project.
