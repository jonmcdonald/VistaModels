vista_rebuild_tree

This script performs a batch rebuild of the specified project and its dependencies. It first deletes all objects (.o files) of the involved projects, then recompiles all sources and builds the design.

Syntax

vista_rebuild_tree [-wd working_directory] project_name

Arguments
  • -wd working_directory
Specifies an existing working directory. (The default working directory is the one that was used when Vista was last invoked in the interactive mode after changes were made to the specified project.)
  • project_name
Name of a Vista project.
