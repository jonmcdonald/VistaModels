Saving a Set of Breakpoints

All of the breakpoints currently set for a design, whether enabled or disabled, can be saved as a file that can later be loaded. This eliminates the need for manually resetting breakpoints every time you begin a simulation session on a given design. Because you can create any number of sets of breakpoints for the same design, saving and loading different breakpoint sets can be a valuable debugging tool.

To save the currently-defined breakpoints as a set:

  1. Choose the Simulation > Breakpoints > Save Breakpoints command.
  1. In the dialog box now displayed, specify the directory location and name of the file that you are creating for this breakpoint set.
