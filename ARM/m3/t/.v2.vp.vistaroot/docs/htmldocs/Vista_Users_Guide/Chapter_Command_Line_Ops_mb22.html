get_most_active_signals

Extracts most active signals from VCD inputs of the model.

Syntax

get_most_active_signals [-h | -help] [-usage] [-wires_included] [-graph_output file-path]
[-output_file file-path] [-percent percent] library_name model_name

Arguments
Common (Optional)
  • [-h | -help]
Provides help.
  • [-usage]
Displays the syntax for this command.
Optional
  • -wires_included
Specifies whether to include wires. Default is regs only.
  • -graph_output file-path
Specifies the path to the graphical output file chained to a VCD file name. The file includes summarized toggle rates of the signals for each VCD file that can be displayed using gnuplot.
  • -output_file file-path
Specifies the path to the output file that should contain the list of most active signals. The default file path is path_to_model_rtl_power_folder/allowed_signals.txt.
  • -percent percent
Specifies the top percent of signals to extract.
Required
  • library_name
Specifies the name of the model library.
  • model_name
Specifies the name of the model.
