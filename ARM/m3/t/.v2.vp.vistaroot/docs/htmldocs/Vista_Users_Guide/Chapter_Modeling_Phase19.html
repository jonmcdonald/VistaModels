FIFO Registers

FIFO (first-in, first out) registers are assigned to a specific address, and during simulation are set and read by other models using a designated port.

Specifying FIFO Registers

To specify FIFO registers for the model:

  1. In Vista's modeling dialog, select the PV tab.
  1. Select the FIFO tab:

Figure 7‑16. Vista Modeling - FIFO Registers Tab

  1. In the Default access port field, select from the pull-down list the port via which registers and memory are accessed. (This can be customized for individual FIFO Registers.)
  1. In the table, enter one or more FIFO registers, specifying the following information for each table entry:
  • Name — Assign a name to each register by entering text.
  • Access port— modify the access port if a value other than the default value is needed
  • Address— assign an address to each register (an address can appear more than once in the table only if the Bit Ranges are different).
  • Width— assign width for the FIFO register. If the value is 0, the width of the access port is used.
  • FIFO Size— assign FIFO size.
