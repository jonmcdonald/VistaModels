Creating Hierarchical Block Diagram's Flow

  1. Open a new block diagram, instantiate components normally and connect them.
  1. Select the appropriate components and use the Add-stub command to create the external sockets (connectors). Using the Add-stub command for a selected component will bind external sockets to any socket-pin of the component that has not already been connected. The new connectors will get the name and side (master/slave) of the component pin.

  Note

While selecting the component, use RMB to select the Add-stub command, to create external connectors for that component. Use the Add-stub command from the Edit icon toolbar on the left to create external connectors for multiple (selected) components instantly.

  1. Save the block diagram.
  1. Open the upper level block diagram and instantiate the lower level block diagram as you do with any other component.

Non-required connectors can be deleted, leaving some component pins open. Connector's names as well as its routings can be changed.
