Graphic Elements

The following paragraphs describe issues related to working with graphic elements in the Block Diagram editor.

Displaying the Grid

The graphical editor positions your elements on a grid. You can make this grid visible by activating the Tools > Options > Setup > Show Gridflag. (Show Grid is a toggle that can be used to make the grid visible or invisible as desired.)

As a shortcut, you can press the F6 function key to alternately display and hide the grid.

the spacing between grid points is 10 pixels by default. You can change this setting by selecting 1 in the “Space between grid points” drop-down list in the Tools > Options > Setup dialog box.

Selecting Elements in a Diagram

You can select elements either individually, or together with other elements.

To select individual elements (and any associated text) —

  1. Activate the Select command icon:

You can do so by (a) clicking it — meaning, as usual, pressing the left mouse button, or by (b) clicking the right mouse button anywhere in the diagram.

  1. Select an element, by clicking the left mouse button. Notice that the element including its accompanying text (if any), changes color to indicate that it is selected.

When you select another element, the previously selected element is deselected.

To select more than one individual element, one by one —

  1. Activate the Select command icon (as above).
  1. While holding down the Shift key, sequentially click on each of the elements that you want to select.

To select a line up to the nearest intersection (node), click once the segment of line that you want to select. To select an entire line up to all endpoints, double-click anywhere on the line.

To select all the elements contained in a rectangular area, as well as their associated text (such as their names) —

  1. Click the left mouse button at one corner of the area, in a place where no element is present.
  1. With the left mouse button still held down, drag the pointer until you've defined a rectangle that completely contains everything you want to select.
  1. Release the mouse button. Notice that the elements change color, indicating that they have been selected.

To select elements based upon predefined criteria —

  1. Click Edit > Query Select in the graphical editor. Or click the right mouse button in an empty area of the diagram, and then, from the popdown that now appears, select the Query Select command.
  1. If you want to select elements found throughout the entire current page, select the Drawing option in the dialog box now displayed. If you want to select a subset of already selected elements, select the Select List option.
  1. Define the elements to be selected using the Select by Name and/or Select by Type controls. For example, if in a block diagram you want to select all the components whose names begin with the letter “M,” select the Blocks check box and enter the string “M*” in the Select by Name text box.

Using the various available check boxes, you can specify as many criteria as you desire. Once the elements you specify are selected, you can manipulate them as you see fit. However, this type of selection is particularly useful in conjunction with defining attributes for multiple graphical elements.

To select all the elements that your diagram contains, use the Select All icon:

Deselecting Elements in a Diagram

To deselect one particular element among those that you have selected in a graphic unit, hold the Shift key down and click that element. —

Everything that you have previously selected is deselected, if you do any of the following —

  • Select anything without depressing the Shift key.
  • Click without holding the Shift key down (that is, click as if to select) in an empty area of the diagram.
Defining Attributes for Graphic Elements

All of the graphic elements used in the various editors have definable attributes. To define or modify these attributes, you use the appropriate dialog box —

  1. Click the Select icon with the left mouse button:

or click anywhere in the diagram with the right mouse button.

  1. Click the element with the left mouse button.
  1. Click the Attributes command icon.

Alternatively, you can hit the F3 key, click Attributes > Element, or select the Attributes command in the popdown displayed when you click the right mouse button.

  1. Define the required attributes using the dialog box now displayed. (For details, see the individual editor chapters.)
  1. After defining the required attributes, save these attributes by clicking Apply (the dialog box remains open) or OK (the dialog box closes). If you click Cancel, the changes you have just made in the dialog box are not saved, and the dialog box closes.

To “page through” the attributes of multiple elements in a diagram —

  1. Select all the elements whose attributes you want to change. (See "Selecting Elements in a Diagram".)
  1. Click Attributes > Element.
  1. Define the required attributes using the dialog box now displayed.
  1. Click the Apply command button. The changes made to attributes in the previously opened dialog box are saved, and the Attributes dialog box for another selected element is automatically opened.
  1. In this way, continue to sequentially modify the attributes for the various elements. When you have made the desired modifications to the last selected element, close the dialog box by clicking OK.
Defining Default Names for Graphic Elements

To define a default name for any or all of the graphic elements of a given type, click Tools > Options > Default Names in the main bar. In the dialog box that now appears, enter the prefix and optional suffix that constitute the default name for each listed element. A running integer is appended to the prefix, when you create the element. Prefixes and suffixes can be defined for components, channels, and other elements.

Specifying Graphic Element Colors

To set default fill and bordering line colors for graphic elements in block diagrams, use the appropriate Tools > Options > Colors panel.

To modify the color of graphic elements locally, select them and click RMB > Colors. In the dialog box now displayed, select fill and line colors.

You can also specify default colors to be displayed when graphic elements are selected with the cursor, when the Attributes dialog box is open, or during a pause in simulation. To do so, open the Tools > Options > Colors dialog box.

Resizing Graphic Elements

To resize a component or other resizable graphic element—

  1. Select the graphic element.
  1. Bring your pointer to the edge or corner that you want to move inward or outward. When your pointer is close enough to the edge or corner, it changes to crosshair shape (+).
  1. While the pointer has its special shape, press and hold down the left mouse button.
  1. Move the pointer toward or away from the center of the graphic element. The edge or corner moves with your pointer, resizing the graphic element.
  1. When the graphic element is the desired size, release the mouse button.

  Note

You can also convert a state's default circular shape into an ellipse in the same way that you resize it.

Moving Graphic Elements

After you place a graphic element in an editing window, you can always move it.
To do so —

  1. Click the Select command icon: or click the right mouse button.
  1. Hold down the left mouse button on the graphic element and drag the element to its new location.
  1. Release the mouse button.

To move several such elements at once —

  1. Select the elements that you want to move.
  1. Hold down the left mouse button on one of the selected elements, and drag them to their new location.
  1. Release the mouse button.
Copying Graphic Elements

In block diagrams, you can copy any graphic element. For the copies, the original names are used when reasonable (for example, the names of instantiated units and components in block diagrams). In cases where you are not likely to want to duplicate the original name, or where you cannot (for example, the name of a state), new default names are automatically assigned. In some cases, while the original name is retained, the name is likely to require editing (such as local connector names). When copying elements, you can use the system clipboard, or bypass the clipboard using a shortcut. (The clipboard, as in many other graphical and text editors, is an off-screen location from which many duplicates can be made. You copy onto the clipboard and paste from the clipboard.)

Using the clipboard, you can copy graphic elements not only within a unit, but also between units of the same type. To do so —

  1. Select the element or elements that you want to copy.
  1. Click the Copy icon: or click Edit > Copy. Alternatively, you can select the Copy command in the popdown that appears when you click the right mouse button.
  1. If you want to copy into another diagram, make that diagram's window active.
  1. Click the Paste icon: or click Edit > Paste. Alternatively, you can select the Paste command in the popdown. The duplicate retrieved from the clipboard moves with your pointer.
  1. Move the duplicate into position and click to drop it. Or, if you decide you don't really want the duplicate, press <Del> on the keyboard or click the right mouse button.

If you want to move duplicated elements again after dropping them, you can move them like any other elements.

If you want another copy of the same element(s), return to the previous step. Your clipboard retains its content until you use the Copy command again or exit Vista. It is separate from the text-editing clipboard.
