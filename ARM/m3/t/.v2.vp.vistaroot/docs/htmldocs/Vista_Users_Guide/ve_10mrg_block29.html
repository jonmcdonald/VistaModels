Updating Component Symbols

If you want to update the component instances in your block diagram to reflect revisions that you have made to the sockets or ports of the instantiated unit, or to the symbol itself —

  1. Choose Tools > Options > Setup and in the dialog box now displayed select the “Enable Update on user defined symbols” checkbox.
  1. If you want any port pins which have been renamed to retain their original positions, select the “Retain pin location during Update/Replace” checkbox.
  1. In the Block Diagram Editor, select the component instances that you want to update.
  1. Click the Update Component command icon: or, in the right-mouse-button, select Update Component.

The component instances that you selected are updated with the latest ports, parameters, and graphics. Where appropriate, the size of the component instance increases to accommodate the additional port/sockets. For instances of the “default” symbol, an automatically updated “default” symbol is used. Instances of other symbols are updated according to your own latest Symbol Editor revisions.
