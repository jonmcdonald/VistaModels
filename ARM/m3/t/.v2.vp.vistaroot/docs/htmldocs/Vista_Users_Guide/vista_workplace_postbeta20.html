Build Menu

Table 3‑5. Build Menu Items 

Item

Description

Compile

Compiles the selected source file.

Abort Compilation

Stops compilation of the selected source file.

Build Project

Performs a build of the project.

Rebuild Project

Deletes the existing object files and rebuilds the project.

Build Executable

Builds an executable for a single sc_main source instead of the entire project.

Relink Executable

Like Build Executable. Can be used after you have modified some libraries or objects linked to an executable.

Build Tree

Builds multiple dependent projects in a bottom-up mode

Rebuild Tree

Deletes the objects of all dependent projects and rebuilds projects in bottom-up mode
