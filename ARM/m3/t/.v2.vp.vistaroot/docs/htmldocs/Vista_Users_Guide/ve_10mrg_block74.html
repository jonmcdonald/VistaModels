Selection Colors

Figure 9‑21. Block Diagram Options - Selection Colors Pane

The Selections Color pane contains the following options:

  • Regular Selection — Define the colors to be displayed in the editors for graphic elements selected using the cursor. Next to each of these items, clicking on the ellipses (...) opens a dialog box list in which the color is defined for that specific item. After you OK the displayed list, the name of the item is displayed in the selected color.
  • Selected with Attributes Dialog Box Open — Define the colors to be displayed for a graphic element when its Attributes dialog box is open.
